# Генератор лабораторных работ «Технологии баз данных»

Этот навык предназначен для генерации уникальных пакетов лабораторных работ (Отчеты + SQL-скрипты + промпты для скриншотов) по дисциплине «Технологии баз данных».

## 1. Входные параметры и метаданные

При генерации необходимо определить две ключевые переменные: **username** (имя пользователя) и **machinename** (название компьютера). Они используются в текстах отчетов, путях к файлам и в промптах для скриншотов.

**Приоритет определения переменных:**
1.  **Явное указание:** Если переменные переданы в текущем запросе.
2.  **Парсинг папки:** Если не указаны, извлечь из названия корневой директории студента в `src/`. Формат названия: `[ID]_[username]_[machinename]` (например, из `2_trushkin_macbus` получаем `trushkin` и `macbus`).
3.  **Рандом:** Если папка называется просто `[ID]`, сгенерировать реалистичные случайные имена.

Дополнительные переменные:
1.  **Операционная система студента:** (Windows, macOS или Linux).
2.  **Фон рабочего стола:** (описание). Ограничение: никаких лишних элементов интерфейса ОС в промптах.

## 2. Алгоритм работы

Для генерации пакета создайте структуру по пути: `src/[Полное_имя_папки_студента]/Технологии баз данных/ЛР_[Номер]/`.

1. **Определение варианта:** Используйте уникальный стиль таблиц, данных и текста.
2. **Генерация отчета:** Создайте Markdown-файл отчета (`Отчет_ЛРX.md`). Структура отчета должна включать: Введение/Цель, Ход работы (содержащий SQL-запросы), Выводы. **Максимизация контента:** Описывайте ход работы максимально подробно. Добавляйте теоретические пояснения к запросам, развернутые описания того, как работает каждая SQL-конструкция, и объемные выводы.
3. **Генерация промптов для скриншотов:** Сгенерируйте промпты для ИИ (на английском языке).
   - **Детализация:** Промпты должны быть максимально подробными (расположение элементов интерфейса PgAdmin, ASCII-таблицы в терминале, шрифты, цвета). Вписывайте точный SQL-код и ответы сервера.
   - **Реализм:** В приглашении командной строки `psql` используйте `username@machinename` или просто `username` в зависимости от ОС.
   - Внедрите входные переменные **OS** и **Фон рабочего стола**.
   - Добавьте строгое ограничение: `The image should look like a direct screen capture showing only the application window (terminal or PgAdmin) over a slightly blurred background of [ФОН]. No desktop icons, no taskbars, no extra OS UI elements.`
4. **Сохранение результатов:** Сохраните сгенерированные файлы `.md`, `.txt` (промпты), `.sql` (скрипты и дампы).
