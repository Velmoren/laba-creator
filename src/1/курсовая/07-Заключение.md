# ЗАКЛЮЧЕНИЕ

В ходе выполнения курсовой работы была спроектирована и реализована база данных для автоматизированной информационной системы управления образовательным центром «Компьютерные курсы».

В результате проведённой работы можно сделать следующие выводы:

1. **Анализ предметной области.** Проведён системный анализ деятельности образовательного центра, определены основные бизнес-процессы, выделены группы пользователей и информационные потоки.

2. **Проектирование базы данных.** Разработана инфологическая модель, содержащая 10 сущностей с атрибутами, определены связи между сущностями. Созданная даталогическая схема приведена к третьей нормальной форме, что обеспечивает отсутствие аномалий при работе с данными.

3. **Программная реализация.** База данных реализована в СУБД PostgreSQL. Созданы все необходимые таблицы с ограничениями целостности, разработаны представления для удобного доступа к данным, реализованы триггеры для автоматизации бизнес-процессов.

4. **Безопасность.** Определена система разграничения доступа с учётом ролей пользователей: администратор, менеджер, преподаватель, бухгалтер.

Разработанная база данных соответствует всем требованиям задания:
- СУБД: PostgreSQL (не Access)
- Количество сущностей: 10 (более требуемых 10)
- Количество триггеров: 4 (более требуемых 3)
- Нормализация: 3НФ

База данных может быть использована как основа для дальнейшей разработки полноценного приложения с пользовательским интерфейсом. Перспективным направлением развития является создание веб-приложения для онлайн-записи на курсы и отслеживания успеваемости слушателями.

---

**Технические требования для развёртывания:**
- СУБД PostgreSQL версии 14 и выше
- Операционная система: Windows/Linux
- Доступ к СУБД с правами администратора

**Рекомендуемое расширение функционала:**
- Добавление модуля онлайн-записи на курсы
- Реализация модуля дистанционного обучения
- Интеграция с платёжными системами
- Мобильное приложение для слушателей
