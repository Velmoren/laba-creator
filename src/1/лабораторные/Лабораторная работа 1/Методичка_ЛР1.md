## Цель работы
Ознакомиться с файловой структурой ОС FreeBSD. Изучить основные команды работы с файлами.

## Теоретические сведения
Во FreeBSD используется иерархическая файловая система. Важной особенностью является разделение базовой системы и стороннего ПО (каталог `/usr/local`).

### Основные команды:
- `mkdir` — создание каталога.
- `ls -la` — просмотр содержимого каталога с атрибутами.
- `cd` — переход между каталогами.
- `rm` — удаление файлов.
- `ln` — создание жёсткой ссылки.
- `ln -s` — создание символической ссылки.

### Ссылки в файловой системе:
1. **Жёсткая ссылка (Hard link)** — дополнительное имя для того же inode. Удаление оригинала не уничтожает данные, пока жива хотя бы одна жёсткая ссылка.
2. **Символическая ссылка (Symlink)** — файл, содержащий путь к другому файлу. Если оригинал удален, ссылка становится «битой».

## Задание
1. Создать два текстовых файла, используя разные текстовые редакторы (`vi` и `nano`).
2. Создать структуру каталогов и файлов, представленную ниже.
3. Создать для файла `HD_D4` одну жёсткую и одну символическую ссылку.
4. Проверить работу ссылок после удаления оригинального файла.

### Структура для создания:
```text
root_dir/
├── D1/
│   └── D3/
│       ├── D6/
│       │   └── HD_D6
│       └── HD_D3
├── D2/
│   ├── D4/
│   │   └── HD_D4 (Оригинал)
│   └── D5/
│       └── HD_D5
└── HD (Файл в корне)
```

## Порядок выполнения
1. Войдите в систему и создайте корневой каталог работы: `mkdir ~/root_dir`.
2. Перейдите в него: `cd ~/root_dir`.
3. Создайте структуру вложенных каталогов: `mkdir -p D1/D3/D6 D2/D4 D2/D5`.
4. Создайте файлы в указанных узлах (например, через `touch` или редакторы).
5. Создайте в `D2/D4` жёсткую ссылку на `HD_D4` с именем `HD1_D4`:
   ```bash
   ln D2/D4/HD_D4 D2/D4/HD1_D4
   ```
6. Создайте символическую ссылку на `HD_D4` в том же каталоге:
   ```bash
   ln -s HD_D4 D2/D4/SL_D4
   ```
7. Выведите дерево файлов командой `ls -R` и проверьте `ls -li` для просмотра номеров inode.
8. Удалите оригинальный файл `HD_D4`. Попробуйте прочитать содержимое `HD1_D4` и `SL_D4`. Сделайте выводы.
