# СКРИНШОТ 10: Зависимость входного сопротивления от частоты

## Имя файла
lr9_var[N]_10_rin_freq.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **ДВА графика Rвх(f) на одном рисунке:**
   - **График 1:** Rвх БЕЗ ООС (низкое значение)
   - **График 2:** Rвх С ООС (высокое значение)
   - Разные цвета для различения

2. **Оси координат:**
   - **Ось X:** Частота в логарифмическом масштабе
     - Диапазон: от 1 Гц до 100 кГц
     - Деления: 1Hz, 10Hz, 100Hz, 1kHz, 10kHz, 100kHz
   
   - **Ось Y:** Входное сопротивление в кОм
     - Диапазон: от 0 до 200 кОм
     - Деления через 20-50 кОм

3. **Характер графиков:**
   
   **БЕЗ ООС (нижняя линия):**
   - Практически горизонтальная линия
   - Значение: около 1-2 кОм
   - Почти не зависит от частоты
   
   **С ООС (верхняя линия):**
   - Горизонтальная линия на более высоком уровне
   - Значение: около 140-150 кОм
   - Также почти не зависит от частоты в рабочем диапазоне

4. **Легенда:**
   ```
   — Без ООС (Rвх ≈ 1.2 кОм)
   — С ООС (Rвх ≈ 145 кОм)
   ```

5. **Отметки значений (курсорами):**
   - Курсор на графике БЕЗ ООС: показать Rвх ≈ 1.2 кОм
   - Курсор на графике С ООС: показать Rвх ≈ 145 кОм

### Как сделать скриншот:

1. **Снять Rвх С ООС:**
   - Установить ключи согласно варианту (с ООС)
   - Analysis → AC
   - Frequency Range: 1 Hz to 100 kHz
   - **Y Expression:** `V(in)/I(V1)`
     - V(in) — напряжение на входе
     - I(V1) — ток через источник V1
     - Отношение дает входное сопротивление
   - Запустить анализ
   - Сохранить результат

2. **Снять Rвх БЕЗ ООС:**
   - Переключить ключи (без ООС)
   - Запустить тот же AC анализ
   - НЕ закрывать окно

3. **Наложить графики:**
   - Plot → Overlay Waveforms
   - Выбрать сохраненный файл с Rвх с ООС
   - Оба графика на одном рисунке

4. **Настроить оси:**
   - Ось Y должна быть в кОм (килоомах)
   - Если в Омах, разделить на 1000
   - Убедиться, что масштаб позволяет видеть оба графика

5. **Добавить легенду:**
   - Plot → Show Legend
   - Подписать графики

6. **Установить курсоры:**
   - Курсор 1: на график БЕЗ ООС (частота ~1 кГц)
   - Курсор 2: на график С ООС (частота ~1 кГц)
   - Считать значения Rвх

7. **Сделать скриншот:**
   - Захватить весь график с легендой
   - Убедиться, что оба графика видны
   - Значения курсоров видны
   - Сохранить как lr9_var[N]_10_rin_freq.png

### Анализ результатов:

**Влияние ООС на входное сопротивление:**

Для последовательной ООС:
```
Rвх_ООС = Rвх × (1 + β×Ku)
145 кОм = 1.2 кОм × (1 + β×165)
1 + β×165 ≈ 120.8
```

Входное сопротивление увеличилось в ~121 раз!

**Физический смысл:**
- ООС создает отрицательную обратную связь по напряжению
- Часть выходного напряжения вычитается из входного
- Для получения того же выходного сигнала нужен больший входной ток
- Результат: входное сопротивление увеличивается

### Что проверить перед сдачей:
- [ ] Два графика на одном рисунке
- [ ] График С ООС значительно выше графика БЕЗ ООС
- [ ] Разница примерно в 100-150 раз
- [ ] Легенда видна
- [ ] Курсоры установлены
- [ ] Значения Rвх видны
- [ ] Оси подписаны (Frequency, Rin)
- [ ] Единицы измерения указаны (кОм)

### Типичные ошибки:
❌ Неправильное выражение Y (не V(in)/I(V1))
❌ Только один график
❌ Графики не различаются (ошибка в измерениях)
❌ Неправильные единицы (Омы вместо кОм)
❌ Rвх зависит от частоты (ошибка в схеме)
❌ Нет легенды

### Практическое значение:

**Высокое входное сопротивление важно для:**
- Уменьшения нагрузки на источник сигнала
- Минимизации искажений входного сигнала
- Согласования с высокоомными источниками
- Каскадного включения усилителей

**Сравнение:**
- **Без ООС:** Rвх = 1.2 кОм
  - Значительная нагрузка на источник
  - Возможны искажения сигнала
  - Ограничения при каскадировании

- **С ООС:** Rвх = 145 кОм
  - Минимальная нагрузка на источник
  - Меньше искажений
  - Легко каскадировать

**Типичные значения Rвх:**
- Микрофонный усилитель: > 10 кОм
- Линейный вход: > 47 кОм
- Инструментальный усилитель: > 1 МОм
- Операционный усилитель: > 1 МОм

Наш усилитель с ООС (145 кОм) подходит для большинства применений!
