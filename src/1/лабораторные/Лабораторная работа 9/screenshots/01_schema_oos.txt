# СКРИНШОТ 1: Схема усилителя с ООС

## Имя файла
lr9_var[N]_01_schema_oos.png
Где [N] — номер варианта (2.1, 2.2, 2.3, 2.4, 2.5)

## Что должно быть на скриншоте

### Обязательные элементы схемы:

1. **Двухкаскадный усилитель:**
   - **Первый каскад:** транзистор Q1 (2N2222)
   - **Второй каскад:** транзистор Q2 (2N2222)
   - Оба транзистора должны быть четко видны

2. **Резисторы с обозначениями и значениями:**
   - R2, R3 — делитель для Q1
   - R4, R5 — резисторы первого каскада
   - R6 — коллекторная нагрузка Q1
   - R7, R8 — резисторы второго каскада
   - R9 — резистор цепи обратной связи (ВАЖНО!)
   - R10 — нагрузка

3. **Конденсаторы:**
   - C1 — входной разделительный
   - C3 — шунтирующий для R3
   - C5, C7 — в цепи обратной связи (для некоторых вариантов)
   - C6 — шунтирующий для R8

4. **Ключи SWITCH (КРИТИЧЕСКИ ВАЖНО!):**
   - SW1, SW2, SW3, SW4, SW5, SW6
   - Должны быть видны ВСЕ ключи
   - Состояние каждого ключа должно быть различимо:
     - Замкнут: линия сплошная или зеленая
     - Разомкнут: линия прерывистая или красная

5. **Цепь обратной связи:**
   - Должна быть четко видна
   - Желательно выделить цветом или обвести
   - Показывает путь сигнала с выхода на вход

6. **Источники:**
   - V1 — источник сигнала
   - V2 = 12 В — источник питания

7. **Узлы схемы:**
   - Входной узел (Vin)
   - Выходной узел (Vout)
   - Промежуточные узлы между каскадами

### Состояние ключей для каждого варианта:

**Вариант 2.1 (Межкаскадная частотно-зависимая ООС):**
- Замкнуты: SW2, SW3, SW4, SW6
- Разомкнуты: SW1, SW5
- Цепь ОС: через R9, C5, C7

**Вариант 2.2 (Внешняя частотно-независимая ООС):**
- Замкнуты: SW2, SW3, SW5, SW6
- Разомкнуты: SW1, SW4
- Цепь ОС: через R9

**Вариант 2.3 (ООС по току в первом каскаде):**
- Замкнуты: SW2, SW3
- Разомкнуты: SW1, SW4, SW5, SW6
- Цепь ОС: через R3

**Вариант 2.4 (ООС по току во втором каскаде):**
- Замкнуты: SW1, SW3
- Разомкнуты: SW2, SW4, SW5, SW6
- Цепь ОС: через R8

**Вариант 2.5 (Параллельная ООС по току):**
- Замкнут: SW1
- Разомкнуты: SW2, SW3, SW4, SW5, SW6
- Цепь ОС: через R4 и R6

### Как сделать скриншот:

1. **Открыть схему:**
   - Запустить Micro Cap 9/10
   - Открыть файл "Cхема без обратной связи - L2.cir"

2. **Установить ключи согласно варианту:**
   - Двойной клик на каждом ключе SWITCH
   - Изменить состояние:
     - Для замкнутого: State = CLOSED
     - Для разомкнутого: State = OPEN
   - Проверить ВСЕ 6 ключей!

3. **Выделить цепь обратной связи (опционально):**
   - Можно обвести цепь ОС красным цветом
   - Или добавить стрелки, показывающие направление сигнала ОС
   - Помогает визуально понять схему

4. **Проверить отображение:**
   - Все элементы видны
   - Обозначения читаемы
   - Состояние ключей различимо
   - Цепь ОС понятна

5. **Сделать скриншот:**
   - Захватить всю схему
   - Убедиться, что все ключи видны
   - Сохранить как lr9_var[N]_01_schema_oos.png

### Подпись под рисунком:
"Рисунок 1 — Схема усилителя с отрицательной обратной связью (вариант [N])"

### Что проверить перед сдачей:
- [ ] Оба транзистора Q1 и Q2 видны
- [ ] Все 6 ключей SWITCH видны
- [ ] Состояние каждого ключа соответствует варианту
- [ ] Цепь обратной связи видна
- [ ] Все резисторы и конденсаторы обозначены
- [ ] Источник питания V2 = 12В виден
- [ ] Входной и выходной узлы обозначены
- [ ] Изображение четкое, надписи читаемы

### Типичные ошибки:
❌ Ключи не переключены (все в состоянии по умолчанию)
❌ Состояние ключей не соответствует варианту
❌ Не видно, какие ключи замкнуты, а какие разомкнуты
❌ Цепь обратной связи не понятна
❌ Схема обрезана, не все элементы видны
❌ Неправильный вариант схемы

### Как проверить правильность варианта:

**Вариант 2.2 (пример):**
1. SW2 замкнут? ✓
2. SW3 замкнут? ✓
3. SW5 замкнут? ✓
4. SW6 замкнут? ✓
5. SW1 разомкнут? ✓
6. SW4 разомкнут? ✓
7. Цепь ОС через R9? ✓

Если все пункты выполнены — вариант правильный!

### Описание цепи ОС в отчете:

Для варианта 2.2 написать:
"Внешняя частотно-независимая отрицательная обратная связь реализована через резистор R9, который соединяет выход усилителя (коллектор Q2) с входом (база Q1). Ключи SW2, SW3, SW5, SW6 замкнуты, ключи SW1, SW4 разомкнуты."
