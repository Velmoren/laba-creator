# СКРИНШОТ 6: Сравнение АЧХ (с ООС и без ООС)

## Имя файла
lr9_var[N]_06_ach_compare.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **ДВА графика АЧХ на одном рисунке:**
   - **Красная кривая:** АЧХ БЕЗ ООС (высокое усиление, узкая полоса)
   - **Синяя кривая:** АЧХ С ООС (низкое усиление, широкая полоса)
   - Разные цвета для четкого различения

2. **Оси координат:**
   - **Ось X:** Частота в логарифмическом масштабе
     - Диапазон: от 0.1 Гц до 100 МГц
     - Деления: 0.1Hz, 1Hz, 10Hz, 100Hz, 1kHz, 10kHz, 100kHz, 1MHz, 10MHz, 100MHz
   
   - **Ось Y:** Коэффициент усиления в дБ
     - Диапазон: от -10 до 50 дБ
     - Деления через 10 дБ

3. **Легенда (обязательно!):**
   - Должна четко указывать:
     ```
     — С ООС (Ku = 1.2, полоса 15 МГц)
     — Без ООС (Ku = 165, полоса 120 кГц)
     ```

4. **Ключевые точки (отметить курсорами или стрелками):**
   
   **Для графика БЕЗ ООС (красный):**
   - Kmax ≈ 44 дБ (165 раз) в области средних частот
   - fнгр ≈ 12 Гц
   - fвгр ≈ 120 кГц
   - Полоса ≈ 120 кГц
   
   **Для графика С ООС (синий):**
   - Kmax ≈ 1.6 дБ (1.2 раза) в области средних частот
   - fнгр ≈ 0.8 Гц
   - fвгр ≈ 15 МГц
   - Полоса ≈ 15 МГц

5. **Визуальное сравнение:**
   - Четко видно, что график С ООС:
     - Ниже по амплитуде (меньше усиление)
     - Шире по частоте (больше полоса)
   - График БЕЗ ООС:
     - Выше по амплитуде (больше усиление)
     - Уже по частоте (меньше полоса)

### Как сделать скриншот:

1. **Сначала снять АЧХ С ООС:**
   - Установить ключи согласно варианту (с ООС)
   - Analysis → AC
   - Frequency Range: 0.1 Hz to 100 MHz
   - Y Expression: db(V(out)/V(in))
   - Запустить анализ
   - Сохранить результат: File → Save Waveform Data

2. **Затем снять АЧХ БЕЗ ООС:**
   - Переключить ключи (SW1,2,3,5 замкнуты; SW4,6 разомкнуты)
   - Запустить тот же AC анализ
   - НЕ закрывать окно с графиком!

3. **Наложить графики:**
   - В Micro Cap: Plot → Overlay Waveforms
   - Выбрать сохраненный файл с АЧХ с ООС
   - Оба графика появятся на одном рисунке
   - Micro Cap автоматически назначит разные цвета

4. **Настроить отображение:**
   - Убедиться, что легенда видна
   - При необходимости: Plot → Show Legend
   - Проверить, что кривые различимы по цвету
   - Добавить подписи к кривым

5. **Добавить аннотации (опционально):**
   - Стрелки с подписями:
     - "Без ООС: Ku=165, полоса=120кГц"
     - "С ООС: Ku=1.2, полоса=15МГц"
   - Помогает наглядно показать разницу

6. **Сделать скриншот:**
   - Захватить весь график с легендой
   - Убедиться, что обе кривые видны
   - Сохранить как lr9_var[N]_06_ach_compare.png

"Рисунок 6 — Сравнение АЧХ усилителя с ООС и без ООС"

### Анализ результатов:

**Влияние ООС на коэффициент усиления:**
```
Ku_ООС = Ku / (1 + β×Ku)
1.2 = 165 / (1 + β×165)
β×165 = 136.5
A = 1 + β×Ku = 137.5
```

**Влияние ООС на полосу пропускания:**
```
Δf_ООС = Δf × A
15 МГц = 120 кГц × 125
```

Теоретически A = 137.5, но полоса расширилась в 125 раз (близко к теории).

### Что проверить перед сдачей:
- [ ] Два графика на одном рисунке
- [ ] Графики разного цвета
- [ ] Легенда видна и понятна
- [ ] Видна разница в усилении (~40 дБ)
- [ ] Видна разница в полосе (~1000 раз)
- [ ] Оси подписаны
- [ ] Логарифмический масштаб по X
- [ ] Диапазон частот достаточен (до 100 МГц)

### Типичные ошибки:
❌ Только один график (не наложены)
❌ Графики одного цвета (не различимы)
❌ Нет легенды
❌ Неправильный диапазон частот (не видно всей полосы с ООС)
❌ Линейный масштаб вместо логарифмического
❌ Графики не совпадают в ОСЧ (ошибка в измерениях)

### Физический смысл:

**Почему усиление уменьшается:**
- ООС вычитает часть выходного сигнала из входного
- Эффективный входной сигнал уменьшается
- Результат: меньше усиление

**Почему полоса расширяется:**
- ООС компенсирует частотные искажения
- На граничных частотах ООС слабее (меньше β)
- Усиление падает медленнее
- Результат: шире полоса

**Произведение усиление×полоса:**
```
Без ООС: 165 × 120 кГц = 19.8 МГц
С ООС: 1.2 × 15 МГц = 18 МГц
```
Произведение примерно сохраняется (закон Боде).

### Практическое значение:

**Когда нужна ООС:**
- Широкополосные усилители
- Стабильные характеристики
- Низкие искажения
- Высокое входное сопротивление

**Когда ООС не нужна:**
- Максимальное усиление
- Узкополосные усилители
- Простые схемы
