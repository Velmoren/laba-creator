## Цель работы

## Теоретические сведения

Отрицательная обратная связь (ООС) — это передача части выходного сигнала усилителя обратно на вход в противофазе с входным сигналом. ООС широко применяется в усилительных устройствах для улучшения их характеристик.

### Виды обратной связи

По способу снятия сигнала обратной связи:
- **Последовательная ООС по напряжению** — сигнал обратной связи пропорционален выходному напряжению
- **Параллельная ООС по току** — сигнал обратной связи пропорционален выходному току

По способу подачи сигнала обратной связи:
- **Последовательная** — сигнал ОС подается последовательно с входным сигналом
- **Параллельная** — сигнал ОС подается параллельно входному сигналу

По частотной зависимости:
- **Частотно-независимая ООС** — глубина обратной связи не зависит от частоты
- **Частотно-зависимая ООС** — глубина обратной связи зависит от частоты

### Влияние ООС на параметры усилителя

1. **Коэффициент усиления**: уменьшается в (1 + βK) раз, где β — коэффициент передачи цепи ОС, K — коэффициент усиления без ОС
2. **Входное сопротивление**: 
   - При последовательной ОС увеличивается в (1 + βK) раз
   - При параллельной ОС уменьшается в (1 + βK) раз
3. **Выходное сопротивление**: уменьшается в (1 + βK) раз
4. **Полоса пропускания**: расширяется в (1 + βK) раз
5. **Нелинейные искажения**: уменьшаются в (1 + βK) раз
6. **Стабильность параметров**: повышается

### Глубина обратной связи

Глубина обратной связи определяется как:
```
A = Kcp / Kcp_ООС
```
где:
- Kcp — коэффициент усиления без ООС в области средних частот
- Kcp_ООС — коэффициент усиления с ООС в области средних частот

## Формулы для предварительного расчета

### Вариант 2.3 и 2.4 (стр. 219)
```
A = 1 + β × K
β = Rэ / (Rэ + rэ)
```
где Rэ = R3 (для варианта 2.3) или Rэ = R8 (для варианта 2.4)

### Варианты 2.1 и 2.2 (стр. 39-42)
```
A = Kcp / Kcp_ООС
Kcp = β × Rк / (rб + rэ)
```

### Вариант 2.5 (стр. 48)
```
A = 1 + β × K
β = R4 / (R4 + Rвх)
```

## Указания к выполнению эксперимента

### Проверка режима по постоянному току
Выполняется с помощью выбора команды **Dynamic DC** в меню **Analysis**. Используйте кнопки пиктограммы на панели инструментов. При их нажатии на схеме появляются значения измеряемых величин.

### Режим расчета частотных характеристик
Для расчета частотных характеристик выберите режим **AC** в меню **Analysis**.

### Режим Stepping
В этом режиме есть возможность снятия графиков при пошаговом изменении величины какого-либо из элементов схемы.

Пример изменения сопротивления нагрузки R10 от 1 кОм до 1.6 кОм с шагом 0.3 кОм.

После снятия графиков делаются выводы о влиянии данного элемента схемы на АЧХ.

**Глубину обратной связи** находим как: A = Ku_cp / Ku_cp_ООС, коэффициенты усиления определяем по соответствующим графикам АЧХ.

### Режим расчета переходных характеристик
Для анализа переходных процессов нужно изменить источник синусоидального напряжения на источник импульсных сигналов (Pulse Source) и изменить параметры модели Pulse.

**Для малых времен:**
- P1 = 0
- P2 = 0.1u
- P3 = 1u
- P4 = 1.1u
- P5 = 10u

**Для больших времен:**
- P1 = 0
- P2 = 0.1m
- P3 = 1m
- P4 = 1.1m
- P5 = 10m

В меню **Analysis** выбрать режим **Transient** и задаться необходимыми параметрами.

## Шаг 1. Предварительный расчет

### 1.1. Расчет коэффициента усиления без ООС

**Коэффициент усиления первого каскада:**
```
K1 = β1 × Rк1 / (rб1 + rэ1) ≈ 75 × 2.2k / (66.7 + 13.9) ≈ 2046
```

**Коэффициент усиления второго каскада:**
```
K2 = β2 × Rк2 / (rб2 + rэ2) ≈ 75 × 1.5k / (66.7 + 13.9) ≈ 1395
```

**Общий коэффициент усиления без ООС:**
```
Kcp = K1 × K2 ≈ 2046 × 1395 ≈ 2,854,170 (нереально высокое значение)
```

*Примечание: В реальности коэффициент усиления будет меньше из-за влияния входных и выходных сопротивлений каскадов*

**Более точный расчет:**
```
Kcp ≈ 100...200 (типичное значение для двухкаскадного усилителя)
```

---

### 1.2. Расчет глубины обратной связи

**Коэффициент передачи цепи ОС:**
```
β = R9 / (R9 + Rвх) ≈ 10k / (10k + 1k) ≈ 0.91
```

**Глубина обратной связи:**
```
A = 1 + β × Kcp = 1 + 0.91 × 150 ≈ 137.5
```

**Коэффициент усиления с ООС:**
```
Kcp_ООС = Kcp / A = 150 / 137.5 ≈ 1.09
```

---

### 1.3. Расчет изменения параметров

**Входное сопротивление:**
```
Rвх_ООС = Rвх × A = 1k × 137.5 = 137.5 кОм
```

**Полоса пропускания:**
```
Δf_ООС = Δf × A = 100 кГц × 137.5 = 13.75 МГц
```
