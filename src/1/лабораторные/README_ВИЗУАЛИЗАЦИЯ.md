## 🎯 Как использовать визуальные материалы

### Вариант 1: Вставить как есть (быстро)

```markdown
1. Открыть файл ВИЗУАЛЬНЫЕ_МАТЕРИАЛЫ.md
2. Скопировать нужный график
3. Вставить в отчет
4. Использовать моноширинный шрифт (Courier New, Consolas)
5. Размер шрифта: 9-10pt
```

**Плюсы:**
- ✅ Быстро (5 минут)
- ✅ Не требует программ
- ✅ Работает в любом редакторе

**Минусы:**
- ❌ Не очень красиво
- ❌ Сложно редактировать

---

### Вариант 2: Перерисовать в Excel (рекомендуется)

```markdown
1. Открыть Excel
2. Создать таблицу с данными из визуализации
3. Вставка → Диаграмма → Точечная (для графиков)
4. Настроить оси, подписи, легенду
5. Экспортировать как изображение (PNG)
```

**Плюсы:**
- ✅ Профессиональный вид
- ✅ Легко редактировать
- ✅ Можно изменить данные

**Минусы:**
- ❌ Требует времени (30-60 минут)
- ❌ Нужен Excel

**Инструкция для Excel:**

#### Для АЧХ/ФЧХ:
1. Создать таблицу:
   ```
   Частота (Гц) | Ku (дБ)
   1            | 20
   10           | 40
   100          | 60
   ...
   ```
2. Выделить данные → Вставка → Диаграмма → Точечная с гладкими кривыми
3. Настроить ось X: логарифмическая шкала
4. Добавить линию -3 дБ (горизонтальная)
5. Подписать граничные частоты

#### Для схем:
1. Использовать Draw.io (diagrams.net)
2. Или Visio / PowerPoint
3. Нарисовать по ASCII-схеме

## 📝 Примеры использования

### Пример 1: АЧХ в Excel

```
Шаг 1: Создать таблицу данных
┌──────────┬─────────┐
│ f, Гц    │ Ku, дБ  │
├──────────┼─────────┤
│ 1        │ 20      │
│ 7.5      │ 63      │ ← fнгр
│ 100      │ 66      │
│ 1000     │ 66      │
│ 10000    │ 66      │
│ 72000    │ 63      │ ← fвгр
│ 1000000  │ 20      │
└──────────┴─────────┘

Шаг 2: Построить график
- Выделить данные
- Вставка → Диаграмма → Точечная
- Ось X: логарифмическая
- Добавить линию на уровне 63 дБ (Kmax - 3 дБ)

Шаг 3: Оформить
- Заголовок: "АЧХ резисторного каскада"
- Ось X: "Частота, Гц"
- Ось Y: "Коэффициент усиления, дБ"
- Добавить стрелки к fнгр и fвгр
```

---

### Пример 2: Схема в Draw.io

```
1. Открыть https://app.diagrams.net
2. Выбрать шаблон "Electrical"
3. Добавить элементы:
   - Резисторы (R1, R2, ...)
   - Конденсаторы (C1, C2, ...)
   - Транзистор (Q1)
   - Источники (V1, V2)
4. Соединить по ASCII-схеме
5. Добавить подписи
6. Экспорт → PNG (300 dpi)
```

---

### Пример 3: Таблица Excel

```
1. Создать структуру как в ВИЗУАЛЬНЫЕ_МАТЕРИАЛЫ.md
2. Ввести формулы:
   - B6: =17*B2+35*B3
   - B9: =66*B2+87*B3
3. Данные → Поиск решения
4. Настроить параметры
5. Скриншот окна (Alt+PrtScr)
6. Вставить в Paint → Сохранить как PNG
```
