# СКРИНШОТ 2: Диалоговое окно "Поиск решения"

## Имя файла
praktikum_lr[N]_task[M]_02_solver.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **Диалоговое окно "Поиск решения" (Solver):**
   - Полное окно должно быть видно
   - Все поля заполнены
   - Кнопки внизу видны

2. **Структура окна:**

```
┌─────────────────────────────────────────────────────┐
│ Поиск решения                                  [X]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│ Установить целевую ячейку:  [$B$6]                 │
│                                                     │
│ Равной:  ⦿ максимальному значению                  │
│          ○ минимальному значению                   │
│          ○ значению:  [        ]                   │
│                                                     │
│ Изменяя ячейки переменных:                         │
│          [$B$2:$B$3]                               │
│                                                     │
│ В соответствии с ограничениями:                    │
│  ┌───────────────────────────────────────────┐     │
│  │ $B$9 <= $D$9                              │     │
│  │ $B$10 <= $D$10                            │     │
│  │ $B$2 >= $D$13                             │     │
│  │ $B$3 >= $D$14                             │     │
│  └───────────────────────────────────────────┘     │
│                                                     │
│ [Добавить] [Изменить] [Удалить] [Восстановить]    │
│                                                     │
│ Выберите метод решения: [Симплекс-метод      ▼]   │
│                                                     │
│ [Параметры] [Найти решение] [Закрыть]             │
└─────────────────────────────────────────────────────┘
```

3. **Целевая ячейка:**
   - Поле: `$B$6`
   - Это ячейка с целевой функцией F
   - Абсолютная ссылка (со знаками $)

4. **Тип оптимизации:**
   - Выбрано: ⦿ максимальному значению
   - Для задачи на максимум прибыли
   - (Для задачи на минимум выбрать "минимальному значению")

5. **Изменяемые ячейки:**
   - Поле: `$B$2:$B$3`
   - Это диапазон с переменными x₁ и x₂
   - Абсолютные ссылки

6. **Ограничения (КРИТИЧЕСКИ ВАЖНО!):**
   - `$B$9 <= $D$9` (расход Р1 ≤ запас Р1)
   - `$B$10 <= $D$10` (расход Р2 ≤ запас Р2)
   - `$B$2 >= $D$13` (x₁ ≥ 0)
   - `$B$3 >= $D$14` (x₂ ≥ 0)
   - Всего 4 ограничения

7. **Метод решения:**
   - Выбрано: "Симплекс-метод"
   - Для линейных задач
   - (Для нелинейных: "Поиск решения ОПГ")

### Как сделать скриншот:

1. **Открыть "Поиск решения":**
   - В Excel: Данные → Поиск решения
   - Если не видно: Файл → Параметры → Надстройки → Поиск решения → Перейти → Поставить галочку

2. **Установить целевую ячейку:**
   - Кликнуть в поле "Установить целевую ячейку"
   - Кликнуть на ячейку B6 в таблице
   - Появится: `$B$6`

3. **Выбрать тип оптимизации:**
   - Для максимизации прибыли: выбрать "максимальному значению"
   - Для минимизации затрат: выбрать "минимальному значению"

4. **Указать изменяемые ячейки:**
   - Кликнуть в поле "Изменяя ячейки переменных"
   - Выделить диапазон B2:B3 в таблице
   - Появится: `$B$2:$B$3`

5. **Добавить ограничения:**
   
   **Ограничение 1 (Ресурс Р1):**
   - Нажать кнопку "Добавить"
   - Ссылка на ячейку: B9
   - Знак: <=
   - Ограничение: D9
   - Нажать "Добавить" (для следующего)
   
   **Ограничение 2 (Ресурс Р2):**
   - Ссылка на ячейку: B10
   - Знак: <=
   - Ограничение: D10
   - Нажать "Добавить"
   
   **Ограничение 3 (x₁ ≥ 0):**
   - Ссылка на ячейку: B2
   - Знак: >=
   - Ограничение: D13 (или просто 0)
   - Нажать "Добавить"
   
   **Ограничение 4 (x₂ ≥ 0):**
   - Ссылка на ячейку: B3
   - Знак: >=
   - Ограничение: D14 (или просто 0)
   - Нажать "OK"

6. **Выбрать метод решения:**
   - Раскрыть список "Выберите метод решения"
   - Выбрать: "Симплекс-метод"
   - Для линейных задач оптимизации

7. **НЕ НАЖИМАТЬ "Найти решение"!**
   - Сначала нужно сделать скриншот
   - Окно должно быть заполнено, но решение еще не найдено

8. **Сделать скриншот:**
   - Захватить всё окно "Поиск решения"
   - Убедиться, что все поля видны и заполнены
   - Сохранить как praktikum_lr1_task1_02_solver.png

### Подпись под рисунком:
"Рисунок 2 — Настройки надстройки «Поиск решения»"

### Что проверить перед сдачей:
- [ ] Целевая ячейка: $B$6
- [ ] Тип оптимизации: максимум (или минимум по условию)
- [ ] Изменяемые ячейки: $B$2:$B$3
- [ ] Все 4 ограничения добавлены
- [ ] Ограничения правильные (знаки <=, >=)
- [ ] Метод решения: Симплекс-метод
- [ ] Окно полностью видно
- [ ] Кнопки внизу видны

### Типичные ошибки:
❌ Целевая ячейка неправильная (не B6)
❌ Неправильный тип оптимизации (минимум вместо максимума)
❌ Изменяемые ячейки неправильные
❌ Не все ограничения добавлены (меньше 4)
❌ Неправильные знаки в ограничениях
❌ Неправильный метод решения (не Симплекс)
❌ Окно обрезано, не все видно
❌ Уже нажата кнопка "Найти решение" (окно закрылось)

### Проверка правильности ограничений:

**Список ограничений должен содержать:**
1. `$B$9 <= $D$9` ✓ (расход Р1 не превышает запас)
2. `$B$10 <= $D$10` ✓ (расход Р2 не превышает запас)
3. `$B$2 >= 0` или `$B$2 >= $D$13` ✓ (x₁ неотрицателен)
4. `$B$3 >= 0` или `$B$3 >= $D$14` ✓ (x₂ неотрицателен)

Если все 4 ограничения есть — правильно!

### Зачем нужен этот скриншот:
- Показать правильность постановки задачи оптимизации
- Доказать, что все ограничения учтены
- Продемонстрировать выбор правильного метода
- Основа для получения решения

### Примечание для nana banana:
Этот скриншот показывает, что вы правильно настроили задачу оптимизации. Преподаватель проверит:
1. Правильность целевой ячейки
2. Тип оптимизации (максимум/минимум)
3. Полноту ограничений (все 4 должны быть!)
4. Выбор метода решения

Без правильных настроек "Поиск решения" не найдет оптимальное решение!
