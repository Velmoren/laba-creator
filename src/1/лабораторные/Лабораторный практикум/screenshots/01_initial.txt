# СКРИНШОТ 1: Исходные данные в Excel

## Имя файла
praktikum_lr[N]_task[M]_01_initial.png
Где [N] — номер ЛР (1 или 2), [M] — номер задачи (1 или 2)

## Что должно быть на скриншоте

### Обязательные элементы:

1. **Рабочий лист Excel с исходными данными:**
   - Название листа видно внизу
   - Лента меню Excel видна сверху
   - Строка формул видна (показывает формулы)

2. **Структура таблицы:**

```
┌───┬──────────────────────┬──────────┬──────────┬──────────┐
│   │          A           │    B     │    C     │    D     │
├───┼──────────────────────┼──────────┼──────────┼──────────┤
│ 1 │ ПЕРЕМЕННЫЕ           │          │          │          │
│ 2 │ x₁ (изделия И1)      │    0     │          │          │
│ 3 │ x₂ (изделия И2)      │    0     │          │          │
│ 4 │                      │          │          │          │
│ 5 │ ЦЕЛЕВАЯ ФУНКЦИЯ      │          │          │          │
│ 6 │ F (прибыль)          │ =17*B2+35*B3        │          │
│ 7 │                      │          │          │          │
│ 8 │ ОГРАНИЧЕНИЯ          │Использов.│ Знак     │ Доступно │
│ 9 │ Ресурс Р1            │=66*B2+87*B3│   ≤    │  12500   │
│10 │ Ресурс Р2            │=84*B2+111*B3│  ≤    │  17100   │
│11 │                      │          │          │          │
│12 │ УСЛОВИЯ              │          │          │          │
│13 │ x₁ ≥ 0               │   =B2    │   ≥      │    0     │
│14 │ x₂ ≥ 0               │   =B3    │   ≥      │    0     │
└───┴──────────────────────┴──────────┴──────────┴──────────┘
```

3. **Ячейки с формулами (КРИТИЧЕСКИ ВАЖНО!):**
   - **B6:** `=17*B2+35*B3` (целевая функция)
   - **B9:** `=66*B2+87*B3` (расход ресурса Р1)
   - **B10:** `=84*B2+111*B3` (расход ресурса Р2)
   - **B13:** `=B2` (условие x₁ ≥ 0)
   - **B14:** `=B3` (условие x₂ ≥ 0)

4. **Строка формул:**
   - Должна быть видна вверху
   - Выделить ячейку B6 (целевая функция)
   - В строке формул должна быть видна формула: `=17*B2+35*B3`
   - Это доказывает, что формулы введены правильно

5. **Начальные значения:**
   - B2 = 0 (x₁)
   - B3 = 0 (x₂)
   - B6 = 0 (F, вычисляется автоматически)
   - B9 = 0 (расход Р1, вычисляется)
   - B10 = 0 (расход Р2, вычисляется)

6. **Подписи и форматирование:**
   - Все ячейки подписаны
   - Заголовки выделены жирным
   - Границы ячеек видны
   - Выравнивание текста правильное

### Как сделать скриншот:

1. **Создать таблицу в Excel:**
   - Открыть новый лист Excel
   - Ввести структуру таблицы как показано выше
   - Назвать лист: "Задача 1.1" (или соответствующий номер)

2. **Ввести формулы (НЕ значения!):**
   - В ячейку B6 ввести: `=17*B2+35*B3`
   - В ячейку B9 ввести: `=66*B2+87*B3`
   - В ячейку B10 ввести: `=84*B2+111*B3`
   - В ячейку B13 ввести: `=B2`
   - В ячейку B14 ввести: `=B3`
   - ВАЖНО: Вводить именно формулы, а не числа!

3. **Ввести начальные значения переменных:**
   - В ячейку B2 ввести: 0
   - В ячейку B3 ввести: 0
   - Формулы автоматически пересчитаются

4. **Ввести ограничения:**
   - В ячейку D9 ввести: 12500
   - В ячейку D10 ввести: 17100
   - В ячейку D13 ввести: 0
   - В ячейку D14 ввести: 0

5. **Показать формулу в строке формул:**
   - Кликнуть на ячейку B6 (целевая функция)
   - В строке формул вверху появится: `=17*B2+35*B3`
   - Это важно для проверки правильности

6. **Отформатировать таблицу:**
   - Выделить заголовки жирным шрифтом
   - Добавить границы ячеек: Главная → Границы → Все границы
   - Выровнять текст по центру в ячейках с данными
   - Установить ширину столбцов для читаемости

7. **Сделать скриншот:**
   - Убедиться, что видны:
     - Лента меню Excel
     - Строка формул с формулой
     - Вся таблица
     - Название листа внизу
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
   - Сохранить как praktikum_lr1_task1_01_initial.png

### Типичные ошибки:
❌ Вместо формул введены числа (например, в B6 написано "0" вместо "=17*B2+35*B3")
❌ Строка формул не видна
❌ Формулы неправильные (опечатки в коэффициентах)
❌ Нет подписей к ячейкам
❌ Таблица не отформатирована (нет границ)
❌ Неправильные значения ограничений
❌ Ячейки B13, B14 содержат числа вместо формул =B2, =B3

### Проверка правильности формул:

**Целевая функция (B6):**
- Кликнуть на B6
- В строке формул должно быть: `=17*B2+35*B3`
- При B2=0, B3=0 результат должен быть 0

**Ограничение Р1 (B9):**
- Кликнуть на B9
- В строке формул должно быть: `=66*B2+87*B3`
- При B2=0, B3=0 результат должен быть 0

**Ограничение Р2 (B10):**
- Кликнуть на B10
- В строке формул должно быть: `=84*B2+111*B3`
- При B2=0, B3=0 результат должен быть 0
