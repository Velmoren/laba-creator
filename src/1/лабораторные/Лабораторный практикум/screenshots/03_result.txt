# СКРИНШОТ 3: Результат решения

## Имя файла
praktikum_lr[N]_task[M]_03_result.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **Рабочий лист Excel с НАЙДЕННЫМ решением:**
   - Та же таблица, что и в скриншоте 1
   - Но теперь с оптимальными значениями
   - Лента меню и строка формул видны

2. **Оптимальные значения переменных:**
   - **B2 (x₁):** оптимальное значение (например, 0)
   - **B3 (x₂):** оптимальное значение (например, 143)
   - Значения могут быть дробными или целыми

3. **Значение целевой функции:**
   - **B6 (F):** максимальная прибыль (например, 5005)
   - Вычислено автоматически по формуле
   - Это ОТВЕТ задачи!

4. **Использование ресурсов:**
   - **B9 (расход Р1):** сколько использовано (например, 12441)
   - **B10 (расход Р2):** сколько использовано (например, 15873)
   - Вычислено автоматически по формулам

5. **Сравнение с ограничениями:**
   - B9 ≤ D9 (12441 ≤ 12500) ✓
   - B10 ≤ D10 (15873 ≤ 17100) ✓
   - Ограничения выполнены!

6. **Дополнительная информация (опционально):**
   - Процент использования ресурсов:
     - Р1: 12441/12500 × 100% = 99.5%
     - Р2: 15873/17100 × 100% = 92.8%

### Как сделать скриншот:

1. **Найти решение:**
   - В окне "Поиск решения" нажать кнопку "Найти решение"
   - Дождаться завершения расчета (обычно несколько секунд)
   - Появится окно "Результаты поиска решения"

2. **Сохранить решение:**
   - В окне "Результаты поиска решения":
   - Выбрать: ⦿ Сохранить найденное решение
   - Можно выбрать отчеты (Результаты, Устойчивость, Пределы)
   - Нажать "OK"

3. **Проверить результат:**
   - Значения в B2 и B3 изменились
   - Значение в B6 показывает максимальную прибыль
   - Значения в B9 и B10 показывают использование ресурсов
   - Все ограничения выполнены

4. **Добавить процент использования (опционально):**
   - В ячейку B13 ввести: `=B9/D9*100`
   - В ячейку B14 ввести: `=B10/D10*100`
   - Формат: Процентный (или добавить знак %)

5. **Выделить ключевые ячейки:**
   - Можно выделить B2, B3, B6 цветом (желтый)
   - Показывает, что это результат решения
   - Помогает визуально

6. **Показать формулу (опционально):**
   - Кликнуть на B6
   - В строке формул видна формула: `=17*B2+35*B3`
   - Показывает, что значение вычислено правильно

7. **Сделать скриншот:**
   - Захватить всю таблицу с результатами
   - Убедиться, что все значения видны
   - Лента меню и строка формул видны
   - Сохранить как praktikum_lr1_task1_03_result.png

### Интерпретация результата:

**Оптимальный план производства:**
- Производить 0 изделий И1
- Производить 143 изделия И2
- Максимальная прибыль: 5005 ден. ед.

**Использование ресурсов:**
- Ресурс Р1: использовано 12441 из 12500 (99.5%)
  - Почти полностью использован (дефицитный)
  - Остаток: 59 единиц
- Ресурс Р2: использовано 15873 из 17100 (92.8%)
  - Есть избыток (недефицитный)
  - Остаток: 1227 единиц

**Вывод:**
- Ресурс Р1 является узким местом (ограничивает производство)
- Увеличение запаса Р1 позволит увеличить прибыль
- Ресурс Р2 в избытке, его увеличение не даст эффекта

### Что проверить перед сдачей:
- [ ] Значения x₁ и x₂ изменились (не 0 и 0)
- [ ] Значение F > 0 (найдена прибыль)
- [ ] Ограничения выполнены (B9 ≤ D9, B10 ≤ D10)
- [ ] Формулы работают (кликнуть на B6, проверить формулу)
- [ ] Все значения видны и читаемы
- [ ] Таблица отформатирована
- [ ] Процент использования вычислен (опционально)

### Типичные ошибки:
❌ Значения остались 0 и 0 (решение не найдено)
❌ Ограничения нарушены (B9 > D9 или B10 > D10)
❌ Значение F = 0 (ошибка в решении)
❌ Дробные значения для целочисленных переменных
❌ Отрицательные значения x₁ или x₂
❌ Формулы заменены на числа (потеряна связь)

### Проверка правильности решения:

**Вручную проверить целевую функцию:**
```
F = 17×x₁ + 35×x₂
F = 17×0 + 35×143 = 5005 ✓
```

**Проверить ограничения:**
```
Р1: 66×0 + 87×143 = 12441 ≤ 12500 ✓
Р2: 84×0 + 111×143 = 15873 ≤ 17100 ✓
x₁ = 0 ≥ 0 ✓
x₂ = 143 ≥ 0 ✓
```

Все ограничения выполнены — решение правильное!
