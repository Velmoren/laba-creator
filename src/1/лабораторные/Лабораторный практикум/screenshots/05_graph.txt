# СКРИНШОТ 5: Графическое решение (для ЛР1, Задача 1)

## Имя файла
praktikum_lr1_task1_05_graph.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **График с осями координат:**
   - **Ось X (горизонтальная):** x₁ (изделия И1)
     - Диапазон: от 0 до 200
     - Деления через 20 или 50
   - **Ось Y (вертикальная):** x₂ (изделия И2)
     - Диапазон: от 0 до 200
     - Деления через 20 или 50

2. **Прямые ограничений:**
   
   **Ограничение 1 (Ресурс Р1):**
   - Уравнение: 66x₁ + 87x₂ = 12500
   - Прямая линия от точки (0, 143.7) до точки (189.4, 0)
   - Подпись: "Р1: 66x₁+87x₂≤12500"
   - Цвет: красный или синий
   
   **Ограничение 2 (Ресурс Р2):**
   - Уравнение: 84x₁ + 111x₂ = 17100
   - Прямая линия от точки (0, 154.1) до точки (203.6, 0)
   - Подпись: "Р2: 84x₁+111x₂≤17100"
   - Цвет: зеленый или оранжевый

3. **Область допустимых решений (ОДР):**
   - Многоугольник, ограниченный прямыми
   - Заштрихован или выделен цветом
   - Вершины:
     - A (0, 0) — начало координат
     - B (189.4, 0) — на оси x₁
     - D (0, 143.7) — на оси x₂
   - Подпись: "ОДР" внутри области

4. **Линии уровня целевой функции:**
   - Несколько параллельных прямых
   - Уравнение: F = 17x₁ + 35x₂ = const
   - Например:
     - F = 2000 (пунктирная)
     - F = 4000 (пунктирная)
     - F = 5005 (жирная, проходит через оптимум)
   - Стрелка, показывающая направление роста F (↗)

5. **Оптимальная точка:**
   - Точка D (0, 143)
   - Отмечена крупным маркером (●)
   - Подпись: "D (0, 143) — ОПТИМУМ"
   - Значение: F = 5005

6. **Вершины ОДР с значениями F:**
   - A (0, 0): F = 0
   - B (189.4, 0): F = 3220
   - D (0, 143): F = 5005 ← максимум

7. **Легенда:**
   - Обозначения линий
   - Цвета ограничений
   - Направление роста F

### Как сделать скриншот:

#### Вариант 1: Построение в Excel

1. **Подготовить данные для графика:**
   
   Создать таблицу точек для прямых:
   ```
   Ограничение Р1:
   x₁ = 0 → x₂ = 12500/87 = 143.7
   x₁ = 189.4 → x₂ = 0
   
   Ограничение Р2:
   x₁ = 0 → x₂ = 17100/111 = 154.1
   x₁ = 203.6 → x₂ = 0
   ```

2. **Создать график:**
   - Вставка → Диаграмма → Точечная с прямыми
   - Добавить ряды данных для каждого ограничения
   - Настроить оси (диапазон 0-200 для обеих)

3. **Добавить прямые ограничений:**
   - Для каждого ограничения создать ряд из 2 точек
   - Соединить прямой линией
   - Назначить цвет и подпись

4. **Выделить ОДР:**
   - Добавить фигуру (многоугольник)
   - Заштриховать или залить цветом (полупрозрачный)
   - Вершины: A, B, D

5. **Добавить линии уровня:**
   - Для F = 5005: 17x₁ + 35x₂ = 5005
   - Точки: (0, 143) и (294.4, 0)
   - Провести прямую
   - Сделать жирной

6. **Отметить оптимум:**
   - Добавить маркер в точке D (0, 143)
   - Сделать крупным и ярким
   - Добавить подпись

7. **Добавить стрелку направления роста F:**
   - Вставка → Фигуры → Стрелка
   - Направление: вверх-влево (↗)
   - Подпись: "Рост F"

#### Вариант 2: Построение в GeoGebra

1. **Открыть GeoGebra:**
   - Онлайн: geogebra.org/graphing
   - Или установленная программа

2. **Ввести уравнения ограничений:**
   - В поле ввода: `66x + 87y = 12500`
   - В поле ввода: `84x + 111y = 17100`
   - Прямые появятся автоматически

3. **Добавить неравенства для ОДР:**
   - `66x + 87y <= 12500`
   - `84x + 111y <= 17100`
   - `x >= 0`
   - `y >= 0`
   - ОДР заштрихуется автоматически

4. **Добавить линию уровня:**
   - `17x + 35y = 5005`
   - Изменить цвет и толщину

5. **Отметить точки:**
   - Ввести: `A = (0, 0)`
   - Ввести: `B = (189.4, 0)`
   - Ввести: `D = (0, 143)`
   - Точки появятся на графике

6. **Настроить вид:**
   - Настройки → Оси → Диапазон 0-200
   - Добавить подписи к точкам
   - Добавить легенду

7. **Экспортировать:**
   - Файл → Экспорт → Изображение (PNG)
   - Или сделать скриншот

#### Вариант 3: Рисование от руки (на бумаге)

1. **Начертить оси:**
   - Горизонтальная ось x₁ (0-200)
   - Вертикальная ось x₂ (0-200)
   - Деления через 20

2. **Построить прямые:**
   - Р1: от (0, 143.7) до (189.4, 0)
   - Р2: от (0, 154.1) до (203.6, 0)
   - Использовать линейку

3. **Заштриховать ОДР:**
   - Найти пересечение прямых с осями
   - Заштриховать область

4. **Отметить оптимум:**
   - Точка D (0, 143)
   - Обвести кружком

5. **Сфотографировать или отсканировать:**
   - Хорошее освещение
   - Четкое изображение
   - Все линии и подписи видны

### Подпись под рисунком:

### Что проверить перед сдачей:
- [ ] Оси подписаны (x₁, x₂)
- [ ] Обе прямые ограничений построены
- [ ] ОДР заштрихована или выделена
- [ ] Оптимальная точка D отмечена
- [ ] Значение F в точке D указано
- [ ] Линия уровня F = 5005 проведена
- [ ] Направление роста F показано
- [ ] Все вершины ОДР обозначены
- [ ] Легенда присутствует

### Типичные ошибки:
❌ Неправильно построены прямые (ошибка в расчетах)
❌ ОДР не заштрихована (не понятно, где решения)
❌ Оптимум не отмечен
❌ Нет линии уровня целевой функции
❌ Не указано направление роста F
❌ Масштаб неправильный (не видно всех точек)
❌ Нет подписей к прямым и точкам
