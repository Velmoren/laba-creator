# Технические требования для выполнения лабораторных работ (ОСРВ)

Для успешного выполнения цикла лабораторных работ студенту необходимо подготовить следующую программную среду:

## 1. Операционная система
- **Базовая ОС**: FreeBSD 12.x или 13.x (рекомендуется 64-битная версия).
- **Среда виртуализации**: Oracle VirtualBox (рекомендуется), VMware Workstation или Proxmox.

## 2. Системные настройки
- **Пользователь**: Обычный пользователь (не root) с возможностью перехода в режим суперпользователя через `su` или `sudo`.
- **Сеть**: Настроенный доступ в интернет (через NAT в виртуальной машине) для установки пакетов.
- **Обновления**: Выполнено `freebsd-update fetch install` и `pkg update`.

## 3. Необходимое ПО (Пакеты)
Минимальный набор инструментов:
- **Редакторы**: `vim`, `nano`, `ee` (встроенный).
- **Сборка**: `gcc` или `clang` (входит в базовую систему).
- **Архиваторы**: `tar`, `gzip`, `bzip2`.
- **Прочее**: `bash`, `curl`, `tree`.

## 4. Дополнительные требования
- Образ CD-ROM или ISO-файл для отработки навыков монтирования (ЛР7).
- USB-накопитель (для проброса в ВМ в ЛР7).
