## Теоретические сведения

Резисторный каскад предварительного усиления — это усилительный каскад на биполярном транзисторе с резистивной нагрузкой в цепи коллектора. Основные характеристики каскада:

### Основные параметры:
- **Коэффициент усиления по напряжению (Kн)** — отношение выходного напряжения к входному
- **Сквозной коэффициент усиления (Kскв)** — коэффициент усиления с учетом входного сопротивления источника сигнала
- **Верхняя граничная частота (fвгр)** — частота, на которой коэффициент усиления падает на 3 дБ (до уровня 0.707) в области высоких частот
- **Нижняя граничная частота (fнгр)** — частота, на которой коэффициент усиления падает на 3 дБ в области низких частот
- **Время установления импульса (tуст)** — время, за которое выходной сигнал достигает установившегося значения
- **Спад плоской вершины импульса (Δ)** — относительное изменение амплитуды импульса на его плоской вершине

### Области частот:
1. **Область средних частот (ОСЧ)** — диапазон частот, в котором коэффициент усиления максимален и постоянен
2. **Область высоких частот (ОВЧ)** — диапазон частот выше fвгр, где влияют паразитные емкости транзистора
3. **Область низких частот (ОНЧ)** — диапазон частот ниже fнгр, где влияют разделительные конденсаторы

### Элементы схемы:
- **R6 (Rк)** — резистор в цепи коллектора, определяет коэффициент усиления
- **R3 (Rэ)** — резистор в цепи эмиттера, обеспечивает стабилизацию режима
- **C3** — конденсатор, шунтирующий резистор R3 по переменному току
- **C4** — разделительный конденсатор на входе, влияет на НГЧ
- **C5** — разделительный конденсатор на выходе, влияет на НГЧ
- **R2, R3** — делитель напряжения для задания смещения базы

## Расчетные формулы

### Область средних частот (ОСЧ):
```
Kн = (β × Rк) / (rб + rэ)
Kскв = Kн × Rвх / (Rвх + Rист)
```

### Область высоких частот (ОВЧ):
```
fвгр = 1 / (2π × τуст)
τуст = Cвх × (Rист || Rвх)
```

### Область низких частот (ОНЧ):
```
fнгр = 1 / (2π × C4 × (Rист + Rвх))
Δ = (Ти / τ) × 100%
```

## Указания к выполнению эксперимента

### Проверка режима каскада по постоянному току
Выполняется с помощью выбора команды **Dynamic DC** в меню **Analysis**. Используйте кнопки пиктограммы на панели инструментов. При их нажатии на схеме появляются значения измеряемых величин.

Убедитесь, что транзистор открыт и работает в линейном режиме [LIN]. В противном случае, проверьте схему.

### Режим расчета частотных характеристик
Для расчета частотных характеристик выберите режим **AC** в меню **Analysis**.

Частотный диапазон (Frequency Range) выбирается так, чтобы просмотреть всю АЧХ каскада: V(7)/V(4) = φ(f)

### Режим Stepping
В этом режиме есть возможность снятия графиков при пошаговом изменении величины какого-либо из элементов схемы.

После снятия графиков делаются выводы о влиянии данного элемента схемы на АЧХ.

### Режим расчета переходных характеристик
Для анализа переходных процессов нужно изменить источник синусоидального напряжения на источник импульсных сигналов (Pulse Source) и изменить параметры модели Pulse.

В меню **Analysis** выбрать режим **Transient** и задаться необходимыми параметрами.

## Приложение Б: Пример выполнения (Вариант 1)

### Исходные данные

**Параметры схемы:**
- C4 = 15 мкФ (по варианту)
- R6 = 2.2 кОм (из схемы)
- C5 = 100 пФ (из схемы)
- R1 = 3.3 кОм (из схемы)
- Напряжение питания V2 = 9 В

**Параметры транзистора КТ316А:**
- rб = 66.7 Ом
- h21э = 75
- Ск = 3 пФ
- f = 150 МГц
- h22э = 20 мкСм
- Iко = 1 мА

### Предварительный расчет

#### Расчет режима по постоянному току

**Напряжение на базе (делитель R2-R3):**
```
Uб = V2 × R3 / (R2 + R3) = 9 × 10 / (47 + 10) ≈ 1.58 В
```

**Ток эмиттера:**
```
Iэ ≈ (Uб - Uбэ) / R5 = (1.58 - 0.7) / 0.47 ≈ 1.87 мА
```

**Ток коллектора:**
```
Iк ≈ Iэ = 1.87 мА
```

**Напряжение коллектор-эмиттер:**
```
Uкэ = V2 - Iк×R6 - Iэ×R5 = 9 - 1.87×2.2 - 1.87×0.47 ≈ 4.0 В
```

**Вывод:** Транзистор работает в активном режиме (Uкэ > 0.3 В)

#### Расчет коэффициентов усиления (ОСЧ)

**Входное сопротивление транзистора:**
```
rвх = h21э × rэ = 75 × 26/1.87 ≈ 1042 Ом
```

где rэ = 26 мВ / Iэ(мА)

**Коэффициент усиления по напряжению:**
```
Kн = h21э × Rк / (rб + rэ) = 75 × 2200 / (66.7 + 13.9) ≈ 2046
```

**Сквозной коэффициент усиления:**
```
Kскв = Kн × Rвх / (Rвх + Rист)
```

где Rвх = R2 || R3 || rвх ≈ 8.2 || 10 || 1.042 ≈ 0.85 кОм

```
Kскв = 2046 × 0.85 / (0.85 + 0.6) ≈ 1200
```

#### Расчет граничных частот

**Верхняя граничная частота (ОВЧ):**
```
Свх = Ск × (1 + Kн) + Сбэ ≈ 3 × (1 + 2046) + 10 ≈ 6148 пФ

τуст = Свх × (Rист || Rвх) = 6148×10⁻¹² × (600 || 850) ≈ 2.3 мкс

fвгр = 1 / (2π × τуст) = 1 / (2π × 2.3×10⁻⁶) ≈ 69 кГц
```

**Нижняя граничная частота (ОНЧ):**
```
fнгр = 1 / (2π × C4 × (Rист + Rвх))
fнгр = 1 / (2π × 15×10⁻⁶ × (600 + 850)) ≈ 7.3 Гц
```

#### Расчет переходных характеристик

**Время установления импульса:**
```
tуст = 2.2 × τуст = 2.2 × 2.3 = 5.06 мкс
```

**Спад плоской вершины импульса (при Ти = 0.9 мс):**
```
τ = C4 × (Rист + Rвх) = 15×10⁻⁶ × 1450 = 21.75 мс

Δ = (Ти / τ) × 100% = (0.9 / 21.75) × 100% ≈ 4.1%
```
