# СКРИНШОТ 4: АЧХ при изменении C4

## Имя файла
lr8_var[N]_04_ach_c4.png

## Что должно быть на скриншоте

### Обязательные элементы:

1. **Несколько графиков АЧХ на одном рисунке:**
   - Минимум 3-4 кривых для разных значений C4
   - Каждая кривая должна быть разного цвета
   - Все кривые должны быть четко различимы

2. **Значения C4 для графиков:**
   - C4 = 0.25 мкФ (минимальное)
   - C4 = 125.25 мкФ (среднее)
   - C4 = 250.25 мкФ (максимальное)
   - Можно добавить промежуточные значения

3. **Легенда (условные обозначения):**
   - Должна быть видна в углу графика
   - Для каждой кривой указано значение C4
   - Цвет линии соответствует значению C4
   - Пример:
     ```
     — C4 = 0.25µF
     — C4 = 125.25µF
     — C4 = 250.25µF
     ```

4. **Оси координат:**
   - **Ось X:** Частота от 0.1 Гц до 100 кГц (логарифмический масштаб)
   - **Ось Y:** Коэффициент усиления в дБ (0-80 дБ)

5. **Видимый эффект изменения C4:**
   - При увеличении C4 нижняя граничная частота fнгр смещается влево (уменьшается)
   - Верхняя граничная частота fвгр остается неизменной
   - В области средних частот все кривые совпадают

### Как сделать скриншот:

1. **Открыть AC Analysis:**
   - Analysis → AC
   - Настроить параметры как в скриншоте 3

2. **Включить режим Stepping:**
   - В окне AC Analysis найти раздел "Stepping"
   - Поставить галочку "Enable Stepping"

3. **Настроить параметр для изменения:**
   - **Component Name:** C4
   - **Step What:** Component Value
   - **Range:**
     - Start: 0.25u (0.25 мкФ)
     - Stop: 250.25u (250.25 мкФ)
     - Step: 125u (125 мкФ)
   - Будет построено 3 графика

4. **Запустить анализ:**
   - Нажать "Run"
   - Micro Cap построит несколько кривых разными цветами
   - Дождаться завершения расчета

5. **Проверить легенду:**
   - Легенда должна появиться автоматически
   - Если нет: Plot → Show Legend
   - Убедиться, что значения C4 видны

6. **Сделать скриншот:**
   - Захватить весь график с легендой
   - Убедиться, что все кривые различимы
   - Сохранить как lr8_var[N]_04_ach_c4.png

### Анализ результатов:

**Влияние C4 на нижнюю граничную частоту:**
```
fнгр = 1 / (2π × C4 × (Rист + Rвх))
```

- При C4 = 0.25 мкФ: fнгр ≈ 440 Гц (высокая)
- При C4 = 15 мкФ: fнгр ≈ 7.5 Гц (средняя)
- При C4 = 250 мкФ: fнгр ≈ 0.45 Гц (низкая)

**Вывод:**
- С увеличением C4 нижняя граничная частота уменьшается
- Полоса пропускания расширяется в область НЧ
- Верхняя граничная частота не изменяется
- В ОСЧ коэффициент усиления одинаков для всех C4

### Физический смысл:

**Почему C4 влияет на fнгр:**
- C4 — разделительный конденсатор на входе
- Образует RC-цепь с входным сопротивлением
- Постоянная времени: τ = C4 × Rвх
- Чем больше C4, тем больше τ, тем ниже fнгр

**Почему C4 не влияет на fвгр:**
- fвгр определяется паразитными емкостями транзистора
- C4 находится на входе, не влияет на ОВЧ
- Верхняя граничная частота зависит от Ск и Сбэ

**Практическое применение:**
- Для усиления низкочастотных сигналов нужен большой C4
- Для экономии места можно использовать меньший C4
- Компромисс между размером конденсатора и полосой пропускания
