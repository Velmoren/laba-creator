# Лабораторная работа №7 «Монтирование устройств и проверка состояния ФС во FreeBSD»

**Дисциплина:** Операционные системы реального времени
**Кафедра:** Интеллектуальные системы в управлении и автоматизации

## Цель работы
Изучение средств для монтирования различных устройств и проверки состояния этих устройств в ОС FreeBSD.

## Теоретические сведения
Во FreeBSD внешние устройства подключаются к общей иерархии каталогов с помощью операции монтирования.
- `mount` — подключение файловой системы.
- `umount` — отключение файловой системы.
- `df` — вывод информации о свободном месте на смонтированных ФС.
- `du` — оценка занимаемого места файлами и каталогами.
- Устройства обычно находятся в каталоге `/dev` (например, `/dev/da0` для флешки, `/dev/cd0` для CD).

## Задание
1. Смонтировать USB-флешку. Просмотреть содержимое. Скопировать с неё файлы в свой домашний каталог.
2. Смонтировать CD-диск. Просмотреть его содержимое.
3. Проверить результаты монтирования командой `df`.
4. Демонтировать файловые системы на этих устройствах.
5. Проверить результаты демонтирования командой `df`.
6. Изучить работу команды `du`.

## Порядок выполнения
1. Монтирование флешки:
   - Создайте точку монтирования: `mkdir /mnt/usb`.
   - Выполните: `mount -t msdosfs /dev/da0s1 /mnt/usb` (тип ФС зависит от вашей флешки).
2. Монтирование CD:
   - Создайте точку монтирования: `mkdir /mnt/cdrom`.
   - Выполните: `mount -t cd9660 /dev/cd0 /mnt/cdrom`.
3. Проверка места: `df -h`.
4. Анализ каталогов: `du -sh ~/root_dir`.
5. Размонтирование: `umount /mnt/usb`, `umount /mnt/cdrom`.

## Контрольные вопросы
