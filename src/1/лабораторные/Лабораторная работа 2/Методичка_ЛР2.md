# Лабораторная работа №2 «Архивирование и сжатие файлов во FreeBSD»

**Дисциплина:** Операционные системы реального времени
**Кафедра:** Интеллектуальные системы в управлении и автоматизации

## Цель работы
Изучить основные команды для архивирования и разархивирования файлов в ОС FreeBSD.

## Теоретические сведения
Архивация и сжатие — это две разные операции. Архивация объединяет файлы в один (tar), а сжатие уменьшает их объем (gzip, bzip2).

### Основные команды:
- `tar -cf archive.tar files` — создать архив.
- `tar -tf archive.tar` — посмотреть список файлов в архиве.
- `tar -xf archive.tar` — распаковать архив.
- `gzip file` — сжать файл (создает `.gz`).
- `bzip2 file` — сжать файл (создает `.bz2`).
- `gunzip / bunzip2` — распаковка.

## Задание
1. Создать один `tar`-архив для нескольких файлов, находящихся в разных каталогах.
2. Создать один `tar`-архив для всех файлов, находящихся в каталоге.
3. Создать один `tar`-архив для всех файлов, находящихся в каталоге и подкаталогах.
4. Сравнить объем полученных архивов с помощью `ls` и убедиться, что `tar` не сжимает файлы.
5. Посмотреть список файлов в архиве и распаковать его.
6. Сжать один из архивов утилитами `gzip` и `bzip2`, сравнить объем полученных файлов.
7. Разархивировать полученный сжатый файл.

## Порядок выполнения
1. Создайте несколько тестовых файлов в разных папках (используйте структуру из ЛР1).
2. Создайте архив из файлов в разных папках:
   ```bash
   tar -cf multi.tar D1/D3/HD_D3 D2/D4/HD_D4
   ```
3. Создайте архив всего каталога:
   ```bash
   tar -cf dir.tar D1/D3
   ```
4. Создайте архив с подкаталогами (рекурсивно):
   ```bash
   tar -cf full.tar root_dir
   ```
5. Проверьте размеры: `ls -l *.tar`.
6. Сожмите файл `full.tar` двумя способами:
   ```bash
   gzip -k full.tar
   bzip2 -k full.tar
   ```
7. Сравните размеры `full.tar.gz` и `full.tar.bz2`. Сделайте вывод о степени сжатия.

## Контрольные вопросы

2. Какие флаги утилиты `tar` используются для создания и распаковки архива?
3. Какой из алгоритмов (gzip или bzip2) обеспечивает более сильное сжатие?
4. Как добавить файл в уже существующий tar-архив?
5. Можно ли сжать каталог командой `gzip` напрямую?
