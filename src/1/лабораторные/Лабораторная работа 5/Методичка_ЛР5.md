# Лабораторная работа №5 «Программирование на языке оболочки Bash во FreeBSD»

**Дисциплина:** Операционные системы реального времени
**Кафедра:** Интеллектуальные системы в управлении и автоматизации

## Цель работы
Изучить основы программирования на языке Bash и использование системных утилит в сценариях.

## Теоретические сведения
Скрипт Bash — это текстовый файл с командами, начинающийся с `#!/usr/local/bin/bash`.

### Основные конструкции:
- `$1, $2, ...` — аргументы командной строки.
- `read var` — ввод данных.
- `if [ ... ]; then ... fi` — условия.
- `for var in ...; do ... done` — циклы.
- `expr` или `(( ... ))` — арифметика.
- `file` — тип файла.
- `ls -li` — просмотр индексных дескрипторов (inode).

## Задание
Написать скрипты для решения следующих задач:
1. Вывод списка параметров командной строки с указанием номера каждого параметра.
2. Присвоение переменным `A, B, C` значений `10, 100, 200`, вычисление и вывод: `D=(A*2 + B/3)*C`.
3. Формирование файла со списком файлов в домашнем каталоге, вывод его в алфавитном порядке и общего количества файлов.
4. Запрос имени пользователя, сравнение с текущим (`$USER`) и вывод сообщения «верно/неверно».
5. Запрос имени файла и вывод сообщения о его типе.
6. Вывод списка всех файлов в заданном каталоге, созданных в один день.
7. Определение, является ли файл символической ссылкой, и прослеживание всей цепочки ссылок.
8. Подсчет строк с заданным словом в файле, если его тип «ISO-8859 text».
9. Поиск файлов в каталоге, являющихся жесткими ссылками на один и тот же файл (по inode).
10. Вывод имен файлов и подкаталогов заданного пользователя и общее число остальных объектов.
11. Вывод списка каталогов из переменной `PATH` и прав доступа к ним.

## Порядок выполнения
1. Установите bash (если нет): `pkg install bash`.
2. Создайте файлы скриптов: `touch script1.sh ... script11.sh`.
3. Сделайте их исполняемыми: `chmod +x *.sh`.
4. Для задачи 2 используйте: `D=$(((A*2 + B/3)*C))`.
5. Для задачи 9 используйте: `ls -li | awk '{print $1, $9}' | sort | uniq -d -D`.
6. Для задачи 11 используйте: `echo $PATH | tr ':' '\n'`.
