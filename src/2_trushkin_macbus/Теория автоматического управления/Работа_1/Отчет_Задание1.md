## ВВЕДЕНИЕ
Настоящая работа посвящена изучению теоретических основ преобразования Лапласа, являющегося базисным инструментом анализа линейных стационарных систем управления. Переход от временного описания динамических процессов к частотному (операторному) позволяет существенно упростить математический аппарат исследования устойчивости и качества регулирования. Целью работы является закрепление навыков нахождения изображений типовых функций воздействия по Лапласу.

## 1. ТЕОРЕТИЧЕСКИЕ ПОЛОЖЕНИЯ
Преобразование Лапласа — это функциональное преобразование, сопоставляющее оригиналу $x(t)$ изображение $X(s)$ по правилу:

$$X(s) = \int_0^{\infty} x(t) e^{-st} dt$$ (1)

где $s = \sigma + j\omega$ — комплексная переменная;
$x(t)$ — функция-оригинал.

Основные теоремы, используемые в инженерной практике:
1. Теорема линейности: изображение суммы функций равно сумме их изображений.
2. Теорема о дифференцировании оригинала: при нулевых начальных условиях $L[x^{(n)}(t)] = s^n X(s)$.
3. Теорема о запаздывании: $L[x(t-\tau)] = e^{-s\tau} X(s)$.

## 2. ТАБЛИЦА ТИПОВЫХ ПРЕОБРАЗОВАНИЙ
Для автоматизации расчетов в ТАУ используется справочная таблица соответствий оригиналов и изображений.

Таблица 1 — Основные оригиналы и их изображения

## 3. ПРАКТИЧЕСКИЕ ПРИМЕРЫ (ВАРИАНТ 10)
В рамках освоения материала были решены следующие задачи на нахождение изображений.

### Задача 1.1. Найти изображение для функции $x_1(t) = 2 \cdot 1(t)$
В силу свойства однородности:
$$X_1(s) = 2 \cdot L[1(t)] = \frac{2}{s}$$ (2)

### Задача 1.10. Найти изображение для функции $x_2(t) = 1(t) \cdot e^{-2t}$
Используя формулу для экспоненциального сигнала при $a = 2$:
$$X_2(s) = \frac{1}{s + 2}$$ (3)

### Задача 1.12. Найти изображение для сигнала $x_3(t) = 4t$
Применяя изображение степенной функции:
$$X_3(s) = \frac{4}{s^2}$$ (4)
