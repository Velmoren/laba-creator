# ОТЧЕТ ПО РАБОТЕ №2: РАСЧЕТ И АНАЛИЗ ПЕРЕХОДНЫХ ХАРАКТЕРИСТИК ЛИНЕЙНОЙ СИСТЕМЫ

## РЕФЕРАТ
Объект исследования: линейная динамическая система второго порядка. Цель работы: нахождение аналитического решения дифференциального уравнения при единичном ступенчатом воздействии. Работа выполнена с применением аппарата операционного исчисления. Объем отчета соответствует ГОСТ 7.32-2017.

## ВВЕДЕНИЕ
Анализ динамических свойств систем автоматического регулирования (САР) базируется на исследовании их реакции на типовые входные сигналы. В данной работе рассматривается задача определения временной функции выхода $y(t)$ системы по заданному дифференциальному уравнению. Ключевым этапом исследования является разложение изображения выходного сигнала на простейшие дроби для последующего нахождения оригинала по таблицам Лапласа.

## 1. ПОСТАНОВКА ЗАДАЧИ (ВАРИАНТ 10)
Задано дифференциальное уравнение системы второго порядка:
$$y''(t) + 3y'(t) + 2y(t) = 5x(t)$$ (1)

Условия задачи:
1. Входное воздействие: единичная ступенчатая функция $x(t) = 1(t)$.
2. Начальные условия: нулевые ($y(0)=0, y'(0)=0$).
3. Требуется найти аналитическое выражение переходной характеристики $y(t)$ и построить график.

## 2. МАТЕМАТИЧЕСКОЕ РЕШЕНИЕ
### 2.1. Операторная форма описания
Переходим от временных зависимостей к их изображениям по Лапласу. С учетом нулевых начальных условий:
$$s^2 Y(s) + 3s Y(s) + 2Y(s) = 5 X(s)$$ (2)

Изображение входного ступенчатого сигнала: $X(s) = 1/s$.
Подставляя $X(s)$ в уравнение (2) и вынося $Y(s)$ за скобки, получаем:
$$Y(s) = \frac{5}{s(s^2 + 3s + 2)} = \frac{5}{s(s+1)(s+2)}$$ (3)

### 2.2. Метод неопределенных коэффициентов
Для нахождения оригинала разложим дробь (3) на сумму простейших составляющих:
$$\frac{5}{s(s+1)(s+2)} = \frac{A}{s} + \frac{B}{s+1} + \frac{C}{s+2}$$ (4)

Приводя правую часть к общему знаменателю, получим тождество:
$5 = A(s+1)(s+2) + Bs(s+2) + Cs(s+1)$

Для нахождения коэффициентов воспользуемся методом частных значений $s$:
1. Пусть $s = 0$: $5 = A(1 \cdot 2) \Rightarrow 5 = 2A \Rightarrow A = 2.5$.
2. Пусть $s = -1$: $5 = B(-1 \cdot 1) \Rightarrow 5 = -B \Rightarrow B = -5$.
3. Пусть $s = -2$: $5 = C(-2 \cdot -1) \Rightarrow 5 = 2C \Rightarrow C = 2.5$.

Проверка: $A + B + C = 2.5 - 5 + 2.5 = 0$. Коэффициенты найдены верно.

### 2.3. Переход ко временной области (Оригинал)
Используя таблицу обратного преобразования Лапласа, находим $y(t)$:
$$y(t) = 2.5 - 5e^{-t} + 2.5e^{-2t}$$ (5)

## 3. АНАЛИЗ ПЕРЕХОДНОГО ПРОЦЕССА
На основе полученного уравнения (5) построен график. В момент времени $t=0$: $y(0) = 2.5 - 5 + 2.5 = 0$, что подтверждает корректность решения. При $t \rightarrow \infty$ система выходит на установившееся значение $y_{ust} = 2.5$. Процесс является монотонным (апериодическим), так как характеристическое уравнение имеет вещественные отрицательные корни.
