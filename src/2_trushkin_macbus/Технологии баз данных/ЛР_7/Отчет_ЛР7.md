# ОТЧЕТ ПО РАБОТЕ №7: АВТОМАТИЗАЦИЯ ДАННЫХ С ПОМОЩЬЮ ТРИГГЕРОВ BEFORE И AFTER

## ВВЕДЕНИЕ
Триггеры позволяют автоматически реагировать на события вставки, обновления или удаления. В данной работе реализованы механизмы автоматического ценообразования и логистической связности.

## 1. ТРИГГЕР АВТОРАСЧЕТА ЦЕНЫ (BEFORE)
Для обеспечения маржинальности продаж создан триггер, который перед вставкой записи в таблицу TRANSACTION устанавливает цену предложения в два раза выше закупочной цены.

```sql
CREATE FUNCTION fn_calc_asking_price() RETURNS TRIGGER AS $$
BEGIN
    NEW.AskingPrice := NEW.AcquisitionPrice * 2;
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER trg_asking_price
BEFORE INSERT ON TRANSACTION
FOR EACH ROW EXECUTE FUNCTION fn_calc_asking_price();
```
