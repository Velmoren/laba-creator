# Отчет по лабораторной работе №2: Задача об оптимальном использовании ресурсов и Транспортная задача

## 1. Введение
Вторая лабораторная работа посвящена решению специализированных задач линейного программирования: оптимизации производственных процессов молочного завода и классической транспортной задаче по минимизации логистических издержек.

## 2. Задача 1. Оптимизация производства (Вариант 1.5)
### 2.1. Постановка задачи
Молочный завод производит молоко, кефир и сметану. Заданы нормы расхода сырья (молока) и фонда времени оборудования. Прибыль: молоко – 30 руб., кефир – 22 руб., сметана – 136 руб. Завод должен производить не менее 100 т молока. Необходимо максимизировать прибыль.

### 2.2. Математическая модель
Вектор переменных $\bar{x} = (x_1, x_2, x_3)$, где $x_1$ - молоко, $x_2$ - кефир, $x_3$ - сметана (в тоннах).
Целевая функция: $E = 30x_1 + 22x_2 + 136x_3 \rightarrow \max$

Ограничения:
$1.01x_1 + 1.01x_2 + 9.45x_3 \le 136$ (Сырье)
$0.18x_1 + 0.19x_2 \le 21.4$ (Основное оборудование)
$3.25x_3 \le 16.25$ (Автоматы расфасовки)
$x_1 \ge 100$ (План по молоку)
$x_1, x_2, x_3 \ge 0$

### 2.3. Решение в Excel
Решение проводилось с использованием вертикальной компоновки ячеек и функции `=СУММПРОИЗВ()`. Таблицы оформлены в академическом корпоративном стиле.

![Решение задачи завода](prompt_excel_2_1_result.png)

При добавлении жесткого ограничения на выпуск кефира $\ge 10$ т, область допустимых решений сократится. Из-за конкуренции за сырье и машинное время это может привести к снижению максимально возможной суммарной прибыли завода.

## 3. Задача 2. Транспортная задача (Вариант 2.5)
### 3.1. Постановка задачи
Задана матрица тарифов $3 	imes 5$. Предложение (карьеры): $A_1=15, A_2=15, A_3=15$. Потребности (участки): $B_1=11, B_2=11, B_3=11, B_4=16, B_5=11$. Суммарное предложение = 45. Суммарный спрос = 60. Задача является открытой (несбалансированной). Необходимо минимизировать транспортные издержки.

### 3.2. Математическая модель
Введем переменные $x_{ij}$ ($i=\overline{1,3}, j=\overline{1,5}$) — объем перевозки от $i$-го поставщика к $j$-му потребителю.
Целевая функция: $E = \sum_{i=1}^3 \sum_{j=1}^5 c_{ij} x_{ij} \rightarrow \min$

Ограничения:
$\sum_j x_{ij} \le a_i$ (для каждого поставщика)
$\sum_i x_{ij} = b_j$ (для каждого потребителя, балансировка требует введения фиктивного поставщика с нулевыми тарифами).
$x_{ij} \ge 0$.

### 3.3. Моделирование в Excel
Для решения применена матричная структура ячеек. Суммы по строкам и столбцам рассчитывались для обеспечения баланса.

**Анализ дополнительных ограничений:**
Введение запрета на перевозку $A_1 \rightarrow B_2$ реализуется путем установки искусственно высокого (заградительного) тарифа $c_{12} = 9999$. 
Ограничение объема перевозки по маршруту не более 3 тонн добавляется как отдельное ограничение $x_{12} \le 3$ в диалоговом окне "Поиск решения". Эти изменения неизбежно приведут к росту суммарных издержек, так как алгоритм будет вынужден искать субоптимальные обходные пути.
