## 1. BEFORE TRIGGER ДЛЯ ЦЕНЫ
Я настроил триггер, который сам считает цену продажи (AskingPrice = AcquisitionPrice * 2). Это исключает ошибки менеджеров при вводе данных. 

```sql
NEW.AskingPrice := NEW.AcquisitionPrice * 2;
```

## 2. AFTER TRIGGER ДЛЯ ЛОГИСТИКИ
При добавлении новой картины в WORK, база автоматически создает строку в TRANSACTION. Это гарантирует, что каждая работа сразу попадает в учет.
![Демонстрация работы триггеров](prompt_db_7.png)

## ЗАКЛЮЧЕНИЕ
Триггеры — крутой способ заставить базу работать на тебя. Теперь менеджеру галереи не нужно считать цену в уме, база сама удвоит AcquisitionPrice при вставке.
