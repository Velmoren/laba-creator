# ОТЧЕТ ПО РАБОТЕ №2: АНАЛИЗ ДИНАМИЧЕСКИХ ХАРАКТЕРИСТИК ЛИНЕЙНОЙ СИСТЕМЫ УПРАВЛЕНИЯ

## РЕФЕРАТ
Объект исследования: линейная динамическая система с передаточной функцией $W(s) = 2/(s+2)$. Цель работы: нахождение реакции системы на экспоненциальное воздействие $x(t) = e^{-t/3}$. Объем отчета соответствует требованиям ГОСТ 7.32-2017.

## ВВЕДЕНИЕ
В задачах автоматизации важно знать, как поведет себя система при подаче на вход тестового сигнала. В моем техническом дневнике я сегодня фиксирую полный цикл решения задачи анализа динамики. Мы перейдем от дифференциальной формы описания к операторной, найдем изображение выходного сигнала и выполним обратный переход во временную область.

## 1. ПОСТАНОВКА ЗАДАЧИ (ВАРИАНТ 18)
Рассматривается инерционное звено с коэффициентом усиления $k=1$ и постоянной времени $T=0.5$ (что соответствует $W(s) = 2/(s+2)$) при входном воздействии $x(t) = e^{-t/3}$.
Начальные условия: $y(0) = 0$.
Задача: найти аналитическое выражение для выходного сигнала $y(t)$ и построить график переходного процесса.

## 2. МАТЕМАТИЧЕСКОЕ РЕШЕНИЕ
### 2.1. Переход в s-область
Изображение входного сигнала $x(t) = e^{-t/3}$ по таблице Лапласа:
$$X(s) = \frac{1}{s + 1/3}$$ (1)

Изображение выходного сигнала $Y(s)$ определяется как произведение изображения входа на передаточную функцию системы:
$$Y(s) = W(s) \cdot X(s) = \frac{2}{s+2} \cdot \frac{1}{s+1/3} = \frac{2}{(s+2)(s+1/3)}$$ (2)

### 2.2. Метод неопределенных коэффициентов
Разложим выражение (2) на сумму простейших дробей:
$$\frac{2}{(s+2)(s+1/3)} = \frac{A}{s+2} + \frac{B}{s+1/3}$$ (3)

Для нахождения $A$ и $B$ составим уравнение:
$2 = A(s + 1/3) + B(s + 2)$

Метод частных значений:
1. При $s = -2$: $2 = A(-2 + 1/3) \Rightarrow 2 = A(-5/3) \Rightarrow A = -6/5 = -1.2$.
2. При $s = -1/3$: $2 = B(-1/3 + 2) \Rightarrow 2 = B(5/3) \Rightarrow B = 6/5 = 1.2$.

Подставляя коэффициенты в (3), получаем:
$$Y(s) = \frac{1.2}{s+1/3} - \frac{1.2}{s+2}$$ (4)

### 2.3. Нахождение временной зависимости (Оригинала)
Применяя обратное преобразование Лапласа к каждой дроби, получаем итоговую временную функцию:
$$y(t) = 1.2 e^{-t/3} - 1.2 e^{-2t}$$ (5)

## 3. ГРАФИЧЕСКИЙ АНАЛИЗ
На основе уравнения (5) построен график. Начальное значение $y(0) = 1.2 - 1.2 = 0$, что соответствует начальным условиям. Функция описывает нарастание сигнала с последующим затуханием (так как входной сигнал сам является затухающей экспонентой). Пик выходного сигнала достигается в точке, где производная $y'(t) = 0$.

## ЗАКЛЮЧЕНИЕ
Использование аппарата Лапласа позволило аналитически рассчитать реакцию системы. В отличие от ступенчатого воздействия, реакция на экспоненту носит импульсный характер с возвратом к нулю при $t \rightarrow \infty$. Работа оформлена по ГОСТ 7.32-2017.
