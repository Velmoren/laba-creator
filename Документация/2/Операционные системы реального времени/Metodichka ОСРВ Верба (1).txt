	•	Вывести количество пользователей, подключенных к системе.
Выполнение работы.
Для выполнения лабораторной необходимо создать файл с содержанием, например, стихотворения А. С. Пушкина «Осень». 
	Чтобы выполнить задание воспользуемся инструментом grep. Он позволяет найти строку в файле или потоке. Его можно использовать с регулярным выражением, чтобы обеспечить большую гибкость при поиске строк.
Теоретические сведения
	•	^: Регулярное выражение, обозначающее начало строки.
	•	т$: Регулярное выражение, обозначающее строки, оканчивающиеся на букву "т".
	•	tail: Утилита для вывода последних частей файла.
	•	-c х: указывает вывод последних х символов из файла.
	•	grep: Утилита для поиска строк, соответствующих заданному шаблону.
	•	'201': Шаблон для поиска. В данном случае ищем строки, содержащие, например, "201".
	•	query4: Имя файла, в котором ведётся поиск.
	•	awk: Утилита для обработки и анализа текстовых данных.
	•	length($0) >= х: Условие, проверяющее длину строки (переданной как $0). Строки с длиной х и более символов будут выбраны.
Перед выполнением следующих команд следует ставить Пайплайн, передающий вывод одной команды на вход другой.
	•	tac: Утилита, которая выводит строки в обратном порядке
	•	sort: сортирует строки в алфавитном порядке
