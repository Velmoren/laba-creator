## 1. Лабораторная работа №1: Установка СУБД и базовая настройка
**Цель:** Установка PostgreSQL, знакомство с SQL Shell (psql) и PgAdmin, создание базы данных и пользователя.

**Требования и шаги для выполнения:**
1. **Установка PostgreSQL:** Скачать версию 15.2 (или актуальную), установить с параметрами по умолчанию. При установке задать пароль суперпользователя (рекомендуется `postgres`). 
2. **Управление службой:** Ознакомиться с запуском службы `postgresql-15`. Остановить и запустить сервер с помощью ярлыков «Stop Server» / «Start Server».
3. **Работа в SQL Shell (psql):**
   - Создать базу данных `test`: `CREATE DATABASE test;`
   - Подключиться к БД `test`: `\c test`
   - Создать нового пользователя `test_user` с паролем `qwerty`: `CREATE USER test_user PASSWORD 'qwerty';`
   - Проверить список баз (`\l`) и ролей (`\dg`).
4. **Работа в PgAdmin (GUI):**
   - Установить PgAdmin, подключиться к локальному серверу PostgreSQL.
   - В интерфейсе PgAdmin создать:
     - Новую базу данных `test1`.
     - Нового пользователя `test_user1`.
   - Использовать инструмент **ERD Tool** для визуального создания таблицы в базе `test1` (задать атрибуты, типы, связи) и сгенерировать SQL-код для её физического создания.

## 2. Лабораторная работа №2: Создание таблиц, запросов и представлений
**Цель:** Изучение DDL и DML операций, выполнение выборок, использование соединений, агрегатных функций и транзакций в консоли psql.

**Требования и шаги для выполнения:**
Все команды выполняются в SQL Shell (psql). Требуется написать SQL-скрипты для следующих задач:

1. **Создание таблиц:**
   - `courses` (Дисциплины): `c_no` (PK), `title`, `hours`.
   - `students` (Студенты): `s_id` (PK), `name`, `start_year`.
   - `exams` (Экзамены): `s_id` (FK), `c_no` (FK), `score`. Составной PK (`s_id`, `c_no`).
2. **Наполнение таблиц:**
   - Внести минимум по 3 записи в каждую таблицу.
3. **Выборка данных (SELECT):**
   - Простая выборка, использование `DISTINCT`.
   - Фильтрация с помощью `WHERE` (включая логические операторы и `NULL`).
4. **Соединения (JOIN):**
   - Прямое декартово произведение.
   - Внутреннее соединение (`JOIN` / `WHERE`).
   - Внешнее соединение (`LEFT JOIN`).
5. **Подзапросы:**
   - Скалярный подзапрос в `SELECT` и `WHERE`.
   - Использование операторов `IN`, `NOT IN`, `EXISTS`.
6. **Сортировка и Группировка:**
   - `ORDER BY` по нескольким полям (`ASC`, `DESC`).
   - Использование агрегатных функций (`COUNT`, `AVG`).
   - Группировка `GROUP BY` и фильтрация групп `HAVING`.
7. **Модификация данных:**
   - Изменение (`UPDATE`) и удаление (`DELETE`) строк.
8. **Транзакции:**
   - Создать таблицу `groups` (`g_no` PK, `head_s_id` FK).
   - Добавить столбец `g_no` в таблицу `students` (`ALTER TABLE`).
   - Продемонстрировать блок `BEGIN ... COMMIT` (обновление данных в двух связанных таблицах). 
   - Продемонстрировать изоляцию транзакций через два параллельных окна терминала.
9. **Представления:**
   - Создать представление (`CREATE VIEW`), объединяющее данные.
