# Навык «Схемотехника»

Этот навык используется как дополнение к `laba-creator` для расчетов и моделирования электронных схем.

## 1. Техническая база ЛР №1: Резисторный каскад
- **Транзистор:** КТ316А (BF=75, VAF=102, RB=66.7, CJC=3.93E-12).
- **Питание:** 9 В.
- **Варианты:** См. полную таблицу вариантов в справочнике.
- **Вопросы:** Список из 33 вопросов из методички ЛР1 является основным.

## 2. Техническая база ЛР №2: Усилитель с ООС
- **Питание:** 12 В.
- **Типы ООС:** 2.1 (Межкаскадная ЧЗ), 2.2 (Внешняя ЧН), 2.3/2.4 (По току), 2.5 (Параллельная по току).
- **Вопросы:** Список из 30 вопросов из методички ЛР2 является основным.

*Все общие правила оформления, нумерации скриншотов, материализации в `source/` и выноса всех ответов на вопросы в отдельный файл наследуются из главного скилла `laba-creator`.*
