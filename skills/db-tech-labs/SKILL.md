# Навык «Базы данных»

Этот навык используется как дополнение к `laba-creator` для генерации контента по проектированию и администрированию БД.

## 1. Спецификации генерации
Для генерации пакета по БД необходимо:
1. **Определение структуры:** Уникальный набор таблиц и связей для каждого студента.
2. **Генерация SQL:** Создание рабочих `.sql` файлов со скриптами создания таблиц, заполнения данными и запросами.
3. **Промпты для PgAdmin:** Помимо терминала `psql`, необходимо генерировать промпты, описывающие графический интерфейс PgAdmin (дерево объектов, вкладка Query Tool с результатом запроса).

## 2. Параметры БД
- **db_name:** Имя создаваемой базы данных.
- **db_user:** Имя владельца БД.
- **machinename:** Название хоста для подключения.

*Все общие правила оформления, нумерации скриншотов, материализации SQL-скриптов в `source/` и уникальности контента наследуются из главного скилла `laba-creator`.*
