# Навык «ОСРВ FreeBSD»

Этот навык используется как дополнение к `laba-creator` для генерации контента по администрированию FreeBSD.

## 1. Спецификации лабораторных работ
Все работы (ЛР 1-7) должны строго соответствовать методическим указаниям из `references/freebsd_guidelines.md`.

**Темы:**
- **ЛР №1:** Файловая структура (mkdir, ln, vim).
- **ЛР №2:** Архивация (tar, gzip, bzip2).
- **ЛР №3:** Пользователи и права (passwd, group, adduser, chown, chmod).
- **ЛР №4:** Текстовые файлы (grep, tail, sort).
- **ЛР №5:** Программирование bash (11 уникальных скриптов).
- **ЛР №6:** Процессы (ps, top, renice, kill) и расписание (cron).
- **ЛР №7:** Монтирование (mount, umount, df, du).

## 2. Параметры ОСРВ
- **username:** Имя пользователя в системе.
- **machinename:** Название хоста (компьютера).
- **shell:** По умолчанию `/bin/sh` или `/usr/local/bin/bash`.

*Все общие правила оформления, работы с `source/` и скриншотами наследуются из главного скилла `laba-creator`.*
