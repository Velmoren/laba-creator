# ГЕНЕРАЛЬНЫЙ ПЛАН: ПРОГРАММНАЯ ГЕНЕРАЦИЯ DOCX (ГОСТ МТУСИ)

## 1. Конфигурация Движка
- **Библиотека**: `docx` (JavaScript).
- **Стиль по умолчанию**: Times New Roman, 14pt, Интервал 1.5, Абзацный отступ 1.25см.
- **Поля**: Left 30mm, Right 15mm, Top 20mm, Bottom 20mm.

## 2. Логика Сборки (scripts/generator.js)
1. **Титульный лист**: Жестко закодированная структура (центрирование, жирность).
2. **Оглавление**: Генерация поля TOC Word.
3. **Текст**: Парсинг Markdown-заголовков (#, ##) в стили Word Heading 1, Heading 2.
4. **Таблицы**: Создание объектов Table с границами 1pt.

## 3. Режим правок
1. Правка вносится в исходный .md в `./Курсовая_КомпьютерныеКурсы/`.
2. Запускается `node scripts/generator.js`.
3. Результат всегда в `./docs/Курсовая_ФИНАЛ_ГОСТ.docx`.

## 4. Контроль качества
- [ ] Оглавление в начале (после титульника).
- [ ] Шрифт TNR 14 везде.
- [ ] Поля соответствуют ГОСТу.
