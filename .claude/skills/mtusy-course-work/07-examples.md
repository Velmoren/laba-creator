## Пример введения

```
ВВЕДЕНИЕ

Современный мир информационных технологий невозможен без использования 
баз данных. Все информационные системы связаны с функциями 
долговременного хранения и обработки информации, которая является 
фактором, определяющим эффективность любой сферы деятельности.

Рынок риэлтерских услуг характеризуется большим объёмом информации 
о недвижимости, клиентах и сделках. Эффективное управление этими 
данными требует применения современных информационных технологий.

Целью курсовой работы является разработка базы данных для 
автоматизированной информационной системы «Риэлтерская фирма 
«Все для Вас».

Для достижения поставленной цели необходимо решить следующие задачи:
1. Провести системный анализ предметной области
2. Изучить существующие аналоги автоматизированных систем
3. Спроектировать инфологическую и даталогическую модели БД
4. Реализовать базу данных в СУБД PostgreSQL
5. Разработать приложения для работы с базой данных

Объектом исследования является ООО «Риэлтерская фирма «Все для Вас».
Предметом исследования — процессы обработки данных о недвижимости 
и сделках.

При выполнении работы использовались методы системного анализа, 
моделирования данных (ER-моделирование), методы нормализации 
реляционных баз данных.

Курсовая работа состоит из введения, трёх глав, заключения, 
списка использованных источников (17 наименований) и приложений.
```

## Пример оформления SQL-кода

```sql
-- Создание таблицы "Клиент"
CREATE TABLE Клиент (
    id_клиента SERIAL PRIMARY KEY,
    фио VARCHAR(200) NOT NULL,
    телефон VARCHAR(20),
    email VARCHAR(100),
    адрес TEXT,
    дата_регистрации DATE DEFAULT CURRENT_DATE
);

-- Создание таблицы "Заказ"
CREATE TABLE Заказ (
    id_заказа SERIAL PRIMARY KEY,
    дата_заказа DATE NOT NULL DEFAULT CURRENT_DATE,
    сумма DECIMAL(12, 2) DEFAULT 0,
    id_клиента INTEGER REFERENCES Клиент(id_клиента)
);

-- Создание триггера для автоматического расчёта суммы
CREATE OR REPLACE FUNCTION обновить_сумму()
RETURNS TRIGGER AS $$
BEGIN
    UPDATE Заказ
    SET сумма = (
        SELECT COALESCE(SUM(количество * цена), 0)
        FROM Состав_заказа
        WHERE id_заказа = NEW.id_заказа
    )
    WHERE id_заказа = NEW.id_заказа;
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER trg_обновить_сумму
AFTER INSERT OR UPDATE ON Состав_заказа
FOR EACH ROW EXECUTE FUNCTION обновить_сумму();
```
