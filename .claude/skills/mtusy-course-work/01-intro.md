# Введение — Цель и задачи курсовой работы

> **Источник:** `source/methodology.md`

---

## Структура введения

Введение должно содержать:

1. **Актуальность и социальная значимость** рассматриваемой темы
2. **Состояние разработанности** в мировой теории и практике (роль и место АИС и БД в современном мире)
3. **Цель работы**
4. **Задачи работы** (в соответствии с главами)
5. **Методы и инструментарий** (при необходимости)
6. **Структура работы** (перечень структурных элементов)

---

## Как формулировать цель

При определении цели **НЕ используйте** глагол «сделать».

**Используйте глаголы:**
- раскрыть
- определить
- установить
- показать
- выявить
- разработать
- создать
- спроектировать

### Примеры формулировки цели:

```
❌ НЕПРАВИЛЬНО:
Целью курсовой работы является сделать базу данных для...

✓ ПРАВИЛЬНО:
Целью курсовой работы является разработка базы данных для АИС «Название»

✓ ПРАВИЛЬНО:
Целью курсовой работы является проектирование и реализация базы данных 
для автоматизированной системы управления «Название предприятия»

✓ ПРАВИЛЬНО:
Целью курсовой работы является создание информационной системы 
управления базой данных для...
```

---

## Как формулировать задачи

Задачи должны соответствовать главам курсовой работы.

**Формат:** Глагол + что делаем + где/для чего

### Примеры формулировки задач:

```
1. Провести системный анализ предметной области и описать объект автоматизации
2. Изучить существующие аналоги и обосновать выбор направления разработки
3. Спроектировать инфологическую модель базы данных
4. Выполнить даталогическое проектирование и провести нормализацию
5. Реализовать базу данных в выбранной СУБД
6. Разработать приложения для работы с базой данных
```

---

## Краткий обзор литературы (дополнительно)

Допустимо включить краткий обзор литературы (3-4 наименования) с аннотациями.

**Требования к обзору:**
- Показывает знакомство студента со специальной литературой
- Демонстрирует умение систематизировать источники
- Выделяет существенное
- Оценивает ранее сделанное другими исследователями
- Определяет главное в современном состоянии изученности темы

**Важно:** Обзор делается только по вопросам выбранной темы, не по всей проблеме. Включается только та литература, с которой студент ознакомился лично.

## Пример введения

```markdown
# ВВЕДЕНИЕ

Современный мир информационных технологий невозможен без использования 
баз данных. Все информационные системы связаны с функциями 
долговременного хранения и обработки информации, которая является 
фактором, определяющим эффективность любой сферы деятельности.

Актуальность темы курсовой работы обусловлена тем, что 
[конкретная причина актуальности для выбранной предметной области].

Целью курсовой работы является разработка базы данных для 
автоматизированной информационной системы «[Название АИС]».

Для достижения поставленной цели необходимо решить следующие задачи:
1. Провести системный анализ предметной области
2. Обосновать выбор информационных технологий
3. Спроектировать инфологическую и даталогическую модели БД
4. Реализовать базу данных в СУБД PostgreSQL
5. Разработать приложения для работы с базой данных

Объектом исследования является [объект автоматизации].
Предметом исследования — процессы обработки данных в [предметная область].

При выполнении работы использовались методы системного анализа, 
моделирования данных, методы нормализации реляционных баз данных.

Курсовая работа состоит из введения, трёх глав, заключения, 
списка использованных источников и приложений.
```
