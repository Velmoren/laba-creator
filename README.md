# Генератор курсовых работ и лабораторных

Автоматическая генерация документов Word (DOCX) из Markdown для курсовых работ и лабораторных по дисциплинам МТУСИ.

## Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Генерация документов

#### Курсовая работа
```bash
npm run build-docx -- [ID]
# Пример: npm run build-docx -- 1
```
Результат: `dist/[ID]/Курсовая_ФИНАЛ_ГОСТ.docx`

#### Лабораторные работы
```bash
npm run build-labs -- [ID]
# Пример: npm run build-labs -- 1
```
Результат: `dist/[ID]/labs/Лабораторная работа X/`

#### Полная сборка с валидацией
```bash
npm run build-all -- [ID]
```

#### Валидация готовых файлов
```bash
npm run validate -- [ID]
```

## Важно: Обновление оглавления

После открытия курсовой работы в Microsoft Word необходимо обновить оглавление:

1. **Способ 1**: Щелкните правой кнопкой мыши на оглавлении → "Обновить поле" → "Обновить целиком"
2. **Способ 2**: Нажмите `Ctrl+A` (выделить всё) → `F9` (обновить все поля)

## Стандарты оформления

- **Шрифт**: Times New Roman, 14pt
- **Интервал**: 1.5
- **Абзацный отступ**: 1.25 см
- **Поля**: Левое 30мм, Правое 15мм, Верхнее/Нижнее 20мм
- **Заголовки**: Автоматическая нумерация и стили по ГОСТ

## Решение проблем

### Файлы не открываются
Запустите валидацию:
```bash
npm run validate -- [ID]
```

### Оглавление пустое
Обновите поле оглавления в Word (см. раздел "Важно: Обновление оглавления")

### Диаграммы не отображаются
Проверьте подключение к интернету (диаграммы рендерятся через Kroki API)
